# 概念选择方法

## 先定义类型，再谈创意

“动作”“角色扮演”这类大标签不够。至少锁定：主类型、子类型、玩家视角、主要输入、
单次时长、核心受众、目标语言文化市场和明确非类型。类型决定玩家用什么能力解决大多数问题；支持类型
只能增强主循环，不能平分产品重心。

同一玩家幻想能用多种类型表达时，优先选择用最少资产和规则就能让核心判断反复成立
的类型。原作有大量战斗不等于必须做实时动作，模型能生成三维画面也不等于三维最适合
验证概念。

## 对标拆解

对标按设计问题选择，不按题材相似选择：

| 问题 | 应寻找的证据 |
|---|---|
| 核心动作 | 哪款作品已让相似动作产生持续的选择和熟练度差异？ |
| 世界响应 | 哪款作品让工具或能力清楚改变环境、角色或路线？ |
| 关卡结构 | 哪款作品用相近时长教会、组合并检验核心机制？ |
| 市场区隔 | 玩家最可能拿它与什么比较，本作凭什么不是低配复刻？ |
| 文化理解 | 原作文化怎样表达题材，目标语言市场怎样理解同类题材和符号？ |

每款只提炼一条最有价值的原则，写出落地方式和排除项。未经核实的玩法细节不进入
交接。参考作品的专有角色、地图、界面、美术、文本和数值都不是可复用设计资产。

## 硬否决

出现任一情况就重做，而不是靠评分补偿：

- 玩家主要阅读对白或沿固定剧情点击；
- 没有可重复动作、压力、取舍或状态变化；
- 换一层题材皮肤仍是同一个游戏；
- 可玩原型依赖开放世界、多人服务或生产级资产量；
- 依赖复制其他游戏受保护的角色、地图、UI 或素材。

## 比较四件事

1. **原作适配**：机制是否表达原作独特规则或情绪，而非只使用名词。
2. **玩家能动性**：玩家是否反复做有后果的决定并逐步掌握它。
3. **视觉传播**：一张实际游戏画面能否同时说明身份、动作和独特世界。
4. **文化适用**：核心幻想和表达在原作语境中是否准确，在目标市场是否无需错误类比就能读懂。
5. **完成风险**：最小原型能否用现有模型与资产证明最危险的承诺。

优先选择最小但能证明核心幻想的原型。相近方案不需要伪精确排序，直接说明关键
取舍和最大不确定性。
