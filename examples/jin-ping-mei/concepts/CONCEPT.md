# 游戏概念：《金瓶梅·大宅两本账》

## 产品定位

| 项目 | 决策 |
|---|---|
| 主类型 | 回合制家宅经营策略（宅斗／后宫权谋） |
| 子类型 | 不完全情报、隐藏账本、仆役情报网、节令回合、AI 对手各有目标、多结局清算 |
| 视角/呈现 | 明代绣像式宅院剖面图，一屏看尽六房与仆役动线；事件切到堂屋席位图 |
| 平台 | 桌面浏览器（≥1280×800），单人，鼠标优先 |
| 原作语境 | 明代汉语世情小说，写市井商户家宅的名分、银钱、人情与礼法 |
| 目标玩家 | 熟悉宅斗／宫斗题材的中文玩家；不读原著也能上手 |
| 界面语言 | 首发简体中文；其他语言不在第一版范围 |
| 时长 | 约 60-90 分钟完整单人流程（三幕：入门／争锋／星散） |
| 随机性 | 传闻真伪、谋算成败、仆役泄密都有概率，但全程走 seedRNG，同种子同操作可复现 |
| 核心卖点 | 全程给你看错的那本账——你以为在争宠和位分，散场时只清算你偷偷攒下的私房、人情与退路 |
| 市场区隔 | 借宅斗题材心智，但不做抽卡、挂机、长线付费和情色内容；一次性买断式短流程单机策略 |

它是**回合制策略**，不是视觉小说、不是恋爱模拟、不是卡牌构筑。每回合的核心是
「在有限行动里，决定这一步是换明面的体面，还是换暗地的退路」，而这两者几乎总是
互相冲突。

## 内容边界（不可放宽）

原著含大量露骨性描写，本作**完全不改编这部分内容**。「宠」在本作中是一种可见的
资源与社会信号——谁主持中馈、谁的席位靠前、谁得了新缎子、谁被托付铺子的账——
通过席位、赏赐、差事与称谓表达，**不描写、不暗示、不以画面或台词呈现任何性内容**。
所有角色着装完整、体面端庄。这既是内容要求，也是本作的设计取向：原著真正的戏
在名分、银钱与人情，不在床笫。

## 目标语言与文化市场

原作文化语境是明代市井商户家庭的礼法秩序：嫡庶、名分、中馈之权、节令人情往来、
私房钱与嫁妆的产权边界。目标市场是中文单机／独立游戏玩家，他们已经通过宅斗剧和
宫斗手游熟悉这套符号，不需要解释「位分」「体面」「私房」「分家」。

界面保留原作术语（位分、体面、中馈、私房、人情、退路、分家），靠后果和短提示教会
玩家，不替换成他文化的通用「声望／金币／好感度」标签。

## 行业对标

| 角色 | 对标作品 | 已被证明的原则 | 本作落地 | 明确不借 |
|---|---|---|---|---|
| 核心动作 | 《Crusader Kings III》 | 「阴谋」把社交行动做成有进度、有同谋、有暴露风险的可重复动作 | 「谋」类行动累积进度，每个知情仆役都是一条泄密通道，风声越高越容易被反制 | 地图征服、实时推进、大战略体量、继承法与封建体系 |
| 隐藏崩塌 | 《王权 Reigns》 | 多条可见指标互相拉扯，玩家为保一条而牺牲另一条 | 明账（位分／体面／宠）与暗账（私房／人情／退路）互为代价，几乎没有双赢动作 | 左右滑牌的浅输入、纯随机事件流、王国题材 |
| 不完全情报 | 《The Shrouded Isle》 | 角色的关键属性默认不可见，只能花代价调查，且线索可能是错的 | 仆役情报网给出带可信度的传闻；证实要花行动与银钱，误信会反噬 | 邪教献祭主题、每周期强制处决一人的结构 |
| 文化市场 | 《中国式家长》 | 中文市场接受「回合制人生 + 多轨数值 + 人情压力」，且愿意为一次性短流程买单 | 节令回合推进十余年家宅生命周期，压力来自礼法与人言而非战斗 | 高考／养娃题材、小游戏合集式的关卡拼贴 |
| 市场反例 | 国产宫斗手游（《宫廷计》类） | 证明宅斗题材在中文市场有稳固心智 | 沿用题材心智与术语体系 | 抽卡、体力、挂机、社交攀比与长线付费；本作是 60-90 分钟一次性单机流程 |

宫斗手游占据的是「长线养成 + 付费加速」的心智。本作不做它的网页缩小版：我们是
一次性通关的策略游戏，赌注是**你会不会在正确的时候放弃争宠**。

最终配方是：**CK3 式的阴谋进度与泄密风险 ＋ Reigns 式的指标互斥 ＋ 不完全情报的
仆役传闻网 ＋ 明代绣像宅院美术 ＋ 忠于原著的西门家兴衰十余年**。

## 方向一：大宅两本账（选定）

**一句话核心卖点**：你扮演带着嫁妆嫁进西门家的孟玉楼，在争宠与攒退路之间反复取舍；
当家主暴亡、大宅星散，游戏只清算你藏起来的那本账。

- **玩家身份**：孟玉楼（三娘）。带布铺资本改嫁入门（第7回），识字、精算、不出头；
  她是原著里少数全身而退的人——第91回改嫁李衙内，带着自己的财产离开。
- **核心动词**：**探**（买通仆役取传闻）、**结**（送礼、援手、结盟）、**谋**（散布、
  截留、揭发，累积进度且有泄密风险）、**持**（管中馈、理铺子账、办节令宴）、
  **藏**（把明面所得换成私房、人情或退路——唯一提高暗账的动作）。
- **回合循环**：进入一个节令 → 读仆役传闻（带可信度）→ 分配有限行动 → 其他五房与
  春梅按各自目标同时行动 → 结算明账／暗账／风声 → 触发本节令事件（宴席席位、
  差事分派、外事冲击）→ 下一节令。
- **三条轴**：
  - **明账**＝位分 × 体面 × 宠。全屏最显眼，有六房排行榜，玩家本能会去争。
  - **暗账**＝私房银 × 人情（谁欠你）× 退路（外部可去处）。默认折叠，不上排行榜。
  - **风声**＝你被怀疑的程度。谋算升风声，风声高则你的传闻被别人买到、你的谋算被反制。
- **世界响应**：五位对手各有独立目标与行动，不是数值墙——潘金莲（第9回入门）好斗且
  会升级冲突；李瓶儿（第19回入门）有钱但脆弱，第30回生下官哥儿成为全宅最高赌注；
  吴月娘握正妻名分与礼法解释权；孙雪娥地位低而怨；庞春梅是可以暴起的通房丫鬟。
  她们互相也在斗，玩家可以借刀。
- **原作转换**：第30回「生子加官」是全宅顶点——有了继承人也有了官身；第59回官哥儿夭、
  第62回李瓶儿死，顶点开始塌；第79回「西门庆贪欲丧命　吴月娘失偶生儿」一章之内
  家主死、遗腹子生，明账在此**一次性清零**；第80-100回分家星散，第87回潘金莲被杀，
  第91回孟玉楼改嫁，第100回孝哥儿被普静度走，家门断绝。
- **最关键的一次反转**：前两幕，UI 把明账做成唯一的赢法，排行榜每回合提醒你落后。
  第79回家主一死，明账整条清零、排行榜消失，结算面板只展开暗账。原著序言写
  「蓋為世戒，非為世勸」——这句话由玩法本身讲出来，不靠旁白。
- **传播画面**：明代绣像式宅院剖面，六房各居一室，仆役提着灯在廊下穿行，玩家看得见
  「谁往谁屋里去」却看不见说了什么；右上角明账排行榜金光闪闪，左下角暗账小字折叠。
- **10-30 分钟原型证明什么**：第一幕（入门）就能证明核心取舍成立——玩家在同一个
  节令里必须在「争一次席位」和「藏一笔私房」之间选，且两者都有可见后果。
- **明确不做**：战斗、恋爱攻略、任何情色内容、开放世界、多人、抽卡、长线养成、
  可改写原著既定死亡的分支（官哥儿、李瓶儿、西门庆、潘金莲的结局不可由玩家逆转）。
- **最大风险**：三条轴若只是三个数字条，玩法会退化成「点最高收益按钮」。必须让
  暗账的价值**在第三幕之前不可见地积累**，且明账的诱惑真实——玩家要在结算时
  真切感到「我早该收手」。
- **最小验证问题**：只做第一幕三个节令 + 一次提前触发的清算，就能验伪最大风险——
  如果试玩者在清算时说「这我早知道」，说明明账诱惑不够；如果说「凭什么」，说明
  暗账线索给得不够。两者都会让我们放弃这个方向。

**三幕节拍**：

- **第一幕·入门（第7-20回，约 20 分钟）**：改嫁入门、认六房、学中馈规矩。教会
  探／结／持／藏四个动作与明暗两账。潘金莲入门（第9回）、李瓶儿入门（第19回）
  各带来一次可读的格局变化。
- **第二幕·争锋（第30-62回，约 30 分钟）**：解锁「谋」与风声。官哥儿降生（第30回）
  把赌注推到最高，全宅目标重排。玩家可以护、可以避、可以借势，但不能亲手加害幼子。
  第59回官哥儿夭折、第62回李瓶儿病故——玩家在此选择揭发真相还是换取好处，
  这一步同时改写人情与风声。
- **第三幕·星散（第79-100回，约 20 分钟）**：第79回家主暴死，明账清零、排行榜消失。
  分家、还债、遣散、各房去向依暗账结算：私房足且退路硬者如孟玉楼改嫁离场（第91回），
  无退路者流落。第100回以孝哥儿被度走收束，家门断绝，无人真正赢过这座宅子。

## 方向二：清河县·铺子账

**类型**：回合制商业与人情经营；账簿与县城地图双面板。

玩家扮演西门家的大掌柜，调度生药铺、缎子铺与解当铺的资金流，用节礼与贿赂打通
提刑所与京中蔡太师门路（第30回买官），同时应付后宅不断抽走现银的压力。核心动词是
「把银子放在哪条关系上」。

- **对标原则**：借经营模拟的现金流压力与关系投资回报，不借其城市建设与科技树。
- **原型范围**：第30回买官前后的一个完整资金周期。
- **淘汰原因**：玩家幻想落在账面调度上，后宅只是抽血的外部压力，恰好丢掉用户要的
  后宫玩法；且核心体验偏表格，视觉传播弱。

## 方向三：雪狮子

**类型**：高概念短体验；七日限时防守与推理。

玩家扮演李瓶儿房里的丫鬟，在官哥儿夭折前的七天内（第59回）布置看护、排查异常、
判断威胁来自哪一房，目标不是抓出凶手，而是让孩子活过第七天。

- **对标原则**：借限时防守中「资源不足以覆盖所有缺口」的紧张感，不借其题材与美术。
- **原型范围**：七天单场景。
- **淘汰原因**：单一事件撑不满 60-90 分钟，且核心体验建立在幼儿之死上，基调过于
  阴冷狭窄；原著中这个结局不可逆转，若允许玩家救下则背离原作，若不允许则玩家
  能动性为零。

## 选择状态

选择 **大宅两本账**。三个方向里只有它同时满足：用户要求的后宫玩法、一个能反复
产生不同结果的核心取舍（明账 vs 暗账）、忠于原著且不依赖情色内容的表达，以及
一个能靠现有资产量完成的 60-90 分钟完整流程。

不可妥协项：

- **两本账的结构不可省**：明账（位分／体面／宠）全程显眼且诱人，暗账（私房／人情／
  退路）全程低调；第79回家主死后明账清零、只清算暗账。这是本作存在的理由。
- **不完全情报**：玩家永远看不到对手的完整状态，只能通过仆役传闻推断，传闻带可信度
  且可能为假；证实要花代价。
- **对手是有目标的行动者**：五房与春梅各有独立目标并互相争斗，玩家可借刀，不做
  只会掉血的数值墙。
- **谋算有暴露代价**：每次谋算累积风声与知情者，泄密会被反制。
- **原著既定结局不可逆转**：官哥儿夭（第59回）、李瓶儿死（第62回）、西门庆死
  （第79回）、潘金莲被杀（第87回）、孝哥儿被度走（第100回）玩家都无法改写；
  玩家能改写的只有自己的去向。
- **内容边界**：不出现任何性描写、露骨台词或暗示性画面；角色着装完整端庄。
- **随机性走 seedRNG**：同种子同操作可复现，自检可重放，但保留传闻真伪与谋算成败的
  概率手感。
- 不做战斗、抽卡、多人、后端、开放世界与长线养成。

## 开放问题

- 第二幕「谋」的行动数量若过多，会稀释每次选择的分量；实装时若一个节令能做超过
  三次谋算，需要收紧行动点。
- 暗账在前两幕的可见度是关键调参点：完全不可见则玩家在清算时感到被欺骗，太可见则
  破坏反转。首版取「可见但不上排行榜、不给出总分」的折中，需试玩确认。
