# 来源设定集：《金瓶梅》

## 来源范围

本分析读取仓库内崇祯本（绣像本）系统百回足本的删节版，不用故事梗概、影视印象或
单个章回代替原著。原文是繁体，策划使用简体；证据统一以回目定位。版本判定、删节
方式和授权边界见 `source/SOURCE.md`。

删节只移除露骨性描写，回目、家宅结构、财物往来、人事调动、诉讼与节令场面全部保留。
本示例的改编方向也不需要被删部分：要提取的是名分、宠、子嗣、钱和信息如何在一所
宅子里流动，不是床笫。

序文自陈全书立意：「蓋為世戒，非為世勸也」「奉勸世人，勿為西門慶之後車，可也」。
这不是外加的道德补丁，而是全书结构本身——一个人靠钱把名分、官职、法律判决和别人的
命都买下来，然后在第79回一次性清算。把这条当作主题脊椎。

## 世界身份

《金瓶梅》不是一部关于某个人的书，而是关于一所宅子的书。西门庆是清河县一个开生药铺
的商人（第1回），靠放官吏债、结姻贵戚、行贿买官，把家产从「饒有幾貫家資」做到
「開四五處鋪面：緞子鋪、生藥鋪、綢絹鋪、絨線鋪」，兼走标船、贩盐引、纳香蜡，
「伙計主管約有數十」，且认蔡太师为干爷（第69回）。

这套系统的真正运行者不是他一个人。宅内六房妻妾各有出身、各有嫁妆、各有私财，
名分排序固定但恩宠可以变动；丫鬟、奶子、家人媳妇、伙计、媒婆、僧尼、帮闲构成
一张贴地的信息网，谁先知道一件事，谁就能改变一件事的结果。全书最值得游戏化的
不是任何一次冲突，而是这个持续运转的结构：**一份有限的注意力、金钱和名分，在多个
有独立意志的人之间反复重新分配，而每次分配都留下不可逆的后果。**

结局给出了这套系统的判决：西门庆33岁暴亡（第79回），亲生子孝哥被普静师幻度出家，
家业由小厮玳安改名西门庆承受（第100回）。血脉断了，宅子还在，继承它的是最会看
风色的下人。

## 可追溯的世界规则

| 规则 | 原作事实 | 游戏化边界 |
|---|---|---|
| 名分是排序，恩宠是流量 | 金莲入门当日磕头递鞋脚、月娘受四礼、众人平叙姊妹之礼、丫头媳妇「趕著他叫五娘」；随后月娘偏爱她，李娇儿等「都氣不忿」（第9回） | 排位固定且公开可见，恩宠每日重新分配；玩家改变的是流量，不是排序 |
| 每房自带独立资本 | 孟玉楼携「現銀子也有上千兩」「南京拔步床也有兩張」「三梭布也有三二百筒」及两个丫鬟入门（第7回）；李瓶儿先后交出三千两及沉香、白蜡、水银、胡椒（第14回、第16回） | 妻妾不是消耗品，是各带资产负债表的行动者；夺其人与夺其财是两件事 |
| 私财只能藏，不能公开持有 | 全书用「梯己」称私房：李瓶儿的梯己之物、韩道国另藏一百两梯己银、送翟管家的梯己三十两（第14回、第70回、第81回） | 私财应是隐藏状态，被发现即引发追索与关系损失 |
| 人有明码标价 | 金莲十五岁被三十两转卖张大户（第1回）；小玉五两、秋菊六两（第9回）；春梅十六两买入、月娘按原价发卖（第9回、第85回）；金莲出宅索价一百两，张二官还八十两不成，武松付满价另与王婆五两（第86-87回）；孙雪娥卖二十五两，薛嫂改契只交八两（第94回） | 人口买卖是这个世界的基础交易，但游戏须呈现其代价而非乐趣，绝不做成收集玩法 |
| 官职可以买，罪名可以撤 | 蔡太师以空名告身札付授西门慶副千户、顶补贺金员缺（第30回）；来保以五百两金银把西门慶从三法司名单上买掉（第18回）；苗青案先行贿平事，被曾御史参劾后再托蔡御史转圜（第47-49回）；敬济以一百两把绞罪改为徒五年（第92回） | 司法结果是可交易资源，须有价格、经手人和被参劾的反向风险，不能只是通关钥匙 |
| 越级会被系统清除 | 宋蕙莲被抬举后自居「爹娘房裡人」与上灶媳妇争执，来旺遭栽赃持刀发配徐州，蕙莲自缢，其父宋仁又被打死（第24-27回） | 下层向上流动是可执行动作，但代价极高且牵连全家，构成高风险高回报支线 |
| 信息由下人传递，且可被截断 | 金莲、李娇儿、春梅、玉楼反复在窗下潜听（第11回、第12回、第20回、第73回）；玳安替西门慶掩饰行踪并向媒婆索钱（第16回、第69回）；月娘的最新消息常经薛嫂转达（第95回） | 偷听、买通、放话应是一等玩家动词，情报改变可用选项而非只加数值 |
| 节令与生日是公开结算日 | 「生日」在全书出现逾百处、涉四十余回；元宵横跨第15、41-46、78-80等回；清明见于第25、48、89、91-92、97回；重阳、中秋、烧纸、念经贯穿始终 | 用节令做回合制时钟：每个节点必须公开分配席位、礼物与体面，把内部排名外化 |
| 子嗣改变一切，但保不住 | 官哥出生同月西门慶得官、月娘受诰（第30回）；官哥受惊夭折（第59回），瓶儿随之而亡（第62回）；孝哥与父同日生，最终被幻度出家（第79回、第100回） | 继承人是最高价值状态，也是最脆弱的状态；不可设计为一次获得即永久持有 |
| 生意靠伙计，伙计会跑 | 各铺本银与掌柜逐一交代（第20回、第79回）；西门慶死后韩道国拐财远遁、来保欺主自开布铺、李娇儿携财归院（第80-81回） | 商业模块的产出应与人事忠诚挂钩；失去人，本金也留不住 |

「私财被发现即触发追索」是从梯己反复被藏、被查、被清点推得的改编规则，不冒充原文
直述。它保留了各房独立经济的真实感，同时给玩家留出隐藏与暴露的操作空间。

## 主要行动者

- **吴月娘**：继室，清河左卫吴千户之女（第1回），持家法与礼制的人。她并非善良的
  背景板：她默许构陷、按原价发卖春梅、逐出金莲（第85-86回），也在第92回亲自去
  县衙告状。终局判词把她列为善终者。
- **李娇儿**：院中出身，第二房。始终与娘家保持通路，西门慶一死即依院中之计携财
  归院（第80回），是「从未真正入局所以能全身而退」的样本。
- **孟玉楼**：第三房，最善判断的人。带巨额嫁妆入门（第7回），西门慶死后主动择嫁
  李衙内并带走全部妆奁与两个陪嫁丫鬟兰香、小鸾（第91回），面对陈敬济的簪子诬敲
  当场变脸、将计就计（第92回）。她是唯一完整带着资本与人手退出的角色。
- **孙雪娥**：先头陈家娘子的陪嫁人，戴鬏髻排行第四（第9回），职权在厨下。名分最
  可疑者最先出局：被拐、受辱、卖入守备府为奴、再被转卖八两（第90回、第94回）。
- **潘金莲**：玩家候选。没有嫁妆、没有娘家、没有子嗣，只有观察力、嘴和对丈夫注意
  力的争夺。她的每一步都是在无资本条件下操作系统：潜听、挑唆、结盟、构陷。她的
  结局证明这条路径在原作规则下不可持续（第87回）。
- **李瓶儿**：第六房，带来全书最大一笔外来资本与唯一的男嗣。她的柔顺换来了宠，
  也使她成为攻击目标；官哥死后她迅速崩溃（第59-62回）。
- **庞春梅**：月娘房里的丫鬟，被拨给金莲「趕著叫娘」（第9回），后被收用（第10回），
  始终无名分。西门慶曾许她「你若到明日有了娃兒，就替你上了頭」（第29回）。被罄身
  发卖后入周守备府，生子、扶正做夫人（第94回），反过来买卖旧主家的人。她是全书
  最完整的位置反转。
- **陈敬济**：女婿，携杨提督案的寄放箱笼进入宅内（第17回），既是资产也是隐患。
  他的一切筹码都是别人的东西，因此每次翻本都更快败落。
- **玳安**：贴身小厮，掌握全部行踪与消息，向媒婆抽成（第69回），最终改名西门庆
  承受家业（第100回）。他证明这个世界真正奖励的是信息位置，不是名分。
- **应伯爵与帮闲**：常设食客，负责撮合生意、引荐门路、放大声势；序文称其为「世之
  小醜」。他们把社交开支变成必需成本。
- **王婆、薛嫂、冯妈妈、文嫂**：媒婆网络，同时经手婚姻、买卖人口、传话与说事，
  在每一次身份变动中抽成。她们是这个世界的交易所。
- **蔡太师、翟管家、夏提刑、周守备、宋巡按**：外部权力层。他们不会主动介入宅内，
  但每一次介入都改写规则；接近他们需要持续的礼物流。

## 全书可重复循环

```text
新的一天/节令到来，宅内注意力与资源重新开盘
  -> 观察谁在何处、谁得了什么、谁少了什么
  -> 从下人、媒婆或帮闲处买到或换到一条信息
  -> 选择动作：争宠、结盟、进言、藏私财、放话、构陷、破财平事
  -> 名分表与恩宠流量发生可见变化，产生新的怨恨与新的债
  -> 外部事件（诉讼、官职、货船、疾病、节令）按其自身节奏落下
  -> 状态被结转到下一轮，旧账不会清零
```

优秀玩家与普通玩家的差别应体现在：是否在动作前买到了信息，是否分得清名分与恩宠、
公产与私财，是否知道哪些人会把话传出去，是否为自己保留了退出资本，以及是否意识到
西门慶这条曲线在第79回一定会断。

## 可玩动词与空间

- 潜听、探问、买通丫鬟与媒婆，把一条私事变成可用的筹码。
- 送礼、设席、安排座次、指定谁陪客，在公开场合完成排名。
- 藏与查私财：把银钱首饰移入箱笼，或清点、追索别人的。
- 进言、挑唆、告状、当众发作，改变第三方对某人的判断。
- 调度人手：把丫鬟从一房拨到另一房、发卖、放出、赎回。
- 破财平事：用钱处理诉讼、参劾、丧仪或一次失控的家丑。
- 在节令与生日上安排出场、衣饰与礼物，把内部地位外化给外人看。

空间语法是同心的，且与名分严格对应：**后边上房**（月娘，全宅信息汇聚处）、
**各房卧房**（各自的私财与秘密所在）、**前边花园与角门**（金莲独居「一個獨獨小角門兒
進去…白日間人跡罕到，極是一個幽僻去處」，第9回；也是全书偷听与私会的主场）、
**厨下**（雪娥的职权，家人媳妇的舆论场）、**门首铺面与卷棚**（男主人与外客的交界）、
**衙门、寺观、坟地、狮子街与临清码头**（外部世界的接口）。这套结构适合一栋可反复
进出的宅子，不需要开放世界。

## 视觉与声音锚点

五间到底七进的门面房与朱红小柜的生药铺；鬏髻、比甲、遍地金衣裙这类把名分穿在身上的
服制；十六两银子买的黑漆欢门描金床与大红罗圈金帐幔（第9回）；账房里的天平、银锭、
文书与合同；解当铺的柜橱与头面匣子；花园角门、粉墙、木香棚与葡萄架的夜色；元宵的
灯楼、烟火与走百病的人流；丧仪的白唐巾、孝绢、五间大棚与传真画像；吴神仙的龟壳扇
与相法；抬进抬出的箱笼——嫁妆抬进来、遗物抬出去用的是同一批箱子；结尾的空宅与
「舊日豪華事已空」。

这些锚点只来自古典文本。任何现代影视、动画或游戏中新增的造型、配色、台词和音乐都
不属于本示例的视觉依据。

## 改编边界

- `immutable`：六房名分次序与各自出身；各房自带嫁妆与私财；官职由行贿取得；苗青案
  与杨提督案中钱改写司法结果；宋蕙莲事件的构陷与后果；官哥与李瓶儿之死；西门慶
  于第79回死、年三十三、与孝哥同日；李娇儿携财归院；孟玉楼带资改嫁并挫败敲诈；
  潘金莲被明码标价后为武松所杀；春梅由婢至守备夫人；孝哥出家、玳安承家、月娘善终。
- `adaptable`：日常场景的时长与密度；宴席与节令的规模；商业往来的具体货品；外部
  官场人物的出场次数；宅内争执的具体口角内容。
- `open`：玩家的每日行动配额、信息的获取成本、私财的隐藏机制、失败与重来的处理、
  是否允许在不改变主要结局的前提下改变某一房的个人结局。
- `conflicted`：作者身份与词话本／崇祯本的文本差异在学术上仍有讨论；本示例只声明
  使用仓库内崇祯本系统删节文本，不把版本判断写进玩法。第7回随孟玉楼入门的小厮
  名琴童，第20回又将李瓶儿带来的天福儿改名琴童，同名不同人，涉及仆役名录时须分辨。

## 风险与设计空白

- 本示例只改编家宅的权力、经济与人情结构。原作的性描写不进入任何策划、界面文案
  与美术资产，也不做暗示性替代。
- 人口买卖、构陷、家暴与自杀是原文事实，必须以后果呈现，不得做成奖励、收集或
  轻佻的玩笑。潘金莲、宋蕙莲、孙雪娥的遭遇应让玩家读到系统之恶，而非获得快感。
- 原作未定义的空白包括：各房日常开支的具体数目、宅内房间的完整平面、丫鬟的全部
  名录与去向、月娘持家的账目细节。这些属于开放设计空间，实现时须标明为原创补全，
  不得冒充原作事实。
- 公版文本不等于所有相关衍生素材可自由使用。实现必须从文字重新设计。
