# 金瓶梅 · 大宅两本账 — 运行与验证

回合制家宅经营策略(宅斗)。你是孟玉楼,带着自己的钱嫁进西门家;
这座宅子会一直告诉你往上爬,它不会告诉你,爬到顶的那天,顶就没了。

## 启动

```bash
cd build/app
python3 -m http.server 5173
# 浏览器打开 http://127.0.0.1:5173/
# 可复现自测:http://127.0.0.1:5173/?seed=42&fast=1
```

零构建、零依赖:原生 ES Module + DOM,桌面浏览器(建议 ≥1280×800)。

## 玩法

- 24 个节令回合、三幕:入门(6)/ 争锋(12)/ 星散(6)。每令 3 点行动,提交后不可撤销。
- 五枚汉字印章即五类行动:
  - **探** — 花银子从仆役(玳安/小玉/冯妈妈/薛媒婆)口中买传闻。传闻带可信度(朱点),可能为假;「证实」再花 1 行动。同一件事不同仆役的说法会矛盾。
  - **结** — 送礼应酬,涨明账(体面/宠)。银子从私房出则掉暗账,从公中出则升风声。
  - **谋** — 第二幕解锁。多回合推进一条谋算(散布流言/夺其差事/坐实传闻/截留书信);每推一次多一名知情者、升一截风声;知情者越多越容易泄密。风声 ≥90 触发「发落」(降位分、罚没私房、禁足一令)——是重挫,不是终局。
  - **持** — 担中馈/铺账/办宴,体面来得最快,还开放公中银;位次太靠后接不到好差事。
  - **藏** — 全局唯一提升暗账的动作:存私房、暗助某人(人情)、开拓退路(娘家/铺子/官媒)。它不涨任何明账,排行榜上你会肉眼可见地掉队。
- 两本账:右上角金色六房排行榜是明账(全场最显眼,是最大的诱饵);左下角墨色小账册是暗账(默认折叠,不上榜,不给总分)。
- 第79回家主暴亡:位分/体面/宠一次性归零,排行榜翻落消失,暗账面板展开占满全屏——此后只清算你藏起来的那本账。结算会显示「历史最高位次」,并明示它对结局贡献为零。
- 五种结局:改嫁李衙内(最佳)/ 归娘家 / 守寡留府 / 遣散流落 / 发落出门,按私房、退路、风声判定。一路追排行榜与早早藏私房,会走向明显不同的结局。
- 存档:每节令自动存入 localStorage,标题画面「续账」继续;「重新过门」清档重开。
- `?seed=N` 固定随机种子:同种子同操作逐字节可复现;`&fast=1` 跳过演出等待(自测用)。

## 音频方案(Web Audio 程序化合成,零素材)

- 五声音阶(宫商角徵羽)+ 丝竹音色:笛/箫(正弦+气声+揉音)、琵琶/古筝(快起音拨弦)、云板/木鱼(短促打击)。散板慢板,无强拍循环。
- BGM 按幕交叉切换:标题(单音笛子长句)→ 第一幕(稀疏拨弦,最轻)→ 第二幕(双声部转厚)→ 第三幕(单声部更低更散)。结局定格一记磬音,改嫁李衙内是一句向上收束的完整乐句,遣散流落中途停住。
- 音效与界面金/墨/朱三色通感:明账增减=清亮金属音,暗账增减=几乎听不见的闷木音,风声升高=压抑的擦弦音;另有云板(提交)、木牌翻动(位次变动)、纸张(传闻)、闷印(证伪「妄」)、堂木(发落)、更漏两下(节令推进)。
- 第79回清零:音乐两秒内抽空、只剩一个衰减低音,然后静默 3-4 秒——全作最重要的听觉时刻,没有音效盖它。
- 开关:局内左上角「音/默」按钮,状态存 localStorage(`jpm_mute`)持久化;首次点击页面后才启动 AudioContext(浏览器自动播放策略)。音频模块独立(`js/audio.js`),删掉不影响玩法。

## 已执行验证

```bash
node test/ledger.mjs        # 纯引擎自测:60 项
python3 test/qa_browser.py  # playwright 全程:38 项断言,控制台 0 报错
```

`test/ledger.mjs`(node,无浏览器,固定种子逐字节可复现)覆盖:
五类行动的收益与代价、明暗两账的不对称(全程扫描:非「藏」不涨暗账、「藏」不涨明账)、
传闻真伪判定与证实、谋算进度/知情者/泄密、风声累积与发落触发、对手每回合行动、
排行榜同量纲(纯藏者掉到榜尾、低位分接不到好差事)、第79回明账清零(归零/撤榜/暗账保留)、
五种结局的判定边界(私房档 × 退路条数逐档)、同种子同操作逐字节一致、存读一致性、
两种策略(追榜 vs 早藏)导向不同结局。

`test/qa_browser.py`(playwright)覆盖:
标题与玩法说明 → 第一幕(过门选择/探/结/藏/传闻卡/假传闻出现/对手入门/明暗不对称)→
第二幕(谋算与风声/铺退路/第59回「换取好处」/第62回「收敛不出头」/李瓶儿撤榜)→
第79回清零演出(演出可观察、归零、排行榜消失、暗账保留、历史最高位次)→ 分家 →
结局(五分支之一、结算点破排行榜无用)→ 第一百回收束(引「蓋為世戒,非為世勸」)→
重开清档。断言控制台 0 报错,过程截图存 `/tmp/jpm_shots/`。
静音开关生效且持久化(`jpm_mute`)也在断言内。
