# 美术方向：《金瓶梅·大宅两本账》

## 视觉定位

明代绣像插图 + 界画（工笔建筑白描）+ 淡彩套印。线描优先、设色克制、宣纸底。
参照对象是明代版画与工笔册页本身，不是任何现代影视、漫画或游戏的宅斗造型。

一句话：**一座画在宣纸上的宅子，你能看见谁往谁屋里去，看不见他们说了什么。**

### 内容边界（最高优先级，压过一切美术考量）

- 不出现任何情色、裸露、暗示性姿态或视线；角色**一律着装完整、体面端庄**，
  明代中上层家常服制，交领／立领，衣不露体。
- 「宠」是社会信号，只用**席位远近、赏赐器物、差事分派、称谓牌**表达，
  不用亲密动作、不用身体接触、不用暧昧构图。
- 卧房场景只画外间与陈设，不进内寝；夜戏用窗纸剪影与灯笼，不画床帏内情形。
- 生成资产时，每一条提示词都必须显式带上「着装完整端庄、无裸露、无暧昧」。

## 核心视觉原则与反向原则

| 核心原则 | 它否决了什么 |
|---|---|
| 情报只由**位置与动线**揭示：谁在谁的院子里、灯往哪走 | 否决在人物头顶挂「好感度」「阴谋进度」浮标 |
| 明账用**金**、暗账用**墨**、风声用**朱**，三色各司其职，不混用 | 否决用同一套彩色数值条表示所有资源；否决把暗账也做成金色 |
| 宅院是**剖面**而非俯视：房间是并置的格子，人在廊下穿行 | 否决 3D 自由视角、否决等距瓦片地图、否决角色在开放世界里走路 |
| 一切细节服务当下决策，宅子会随家道**减细节** | 否决用随机器物堆砌繁荣感；第三幕必须肉眼可见地空下来 |
| 用明代版画的**刻痕与套色错位**统一全部资产 | 否决水墨写意笔触、否决现代插画厚涂、否决 Q 版大头 |

## 镜头与构图

主视图是一张**固定的宅院剖面立面图**，一屏看尽：正房（吴月娘）与五房厢院、
花园、前厅铺面、仪门与影壁。房间是并置的矩形格子，人物是格子里的小立像，
仆役在廊下与院门之间移动——**玩家的核心信息来自「谁在移动、去了哪里」**。

焦点顺序：① 廊下移动的仆役灯火 → ② 右上角金色六房排行榜 → ③ 各房内的人物立像
→ ④ 左下角折叠的墨色暗账。第三幕这个顺序被强制重排（见签名画面三）。

事件切到两种子镜头：

- **席位图**：堂屋俯视，长案与座次一目了然，座次即位分。玩家点座次做选择。
- **对话卡**：人物半身立绘居右，宣纸文本框居左，不遮挡宅院图的关键动线。

遮挡容忍度：人物立像不得遮挡廊道（动线是情报）；文本框不得压住排行榜与暗账面板。

## 世界与角色语法

### 形状与尺度

- **建筑**：直线、对称、灰瓦白墙、朱柱。等级越高的房，开间越宽、台基越高、
  斗拱越繁——位分之差**在建筑上直接可读**。
- **六房**（原著名分次序，不可改）：轮廓必须一眼分得开，靠**发髻形制、外衣廓形、
  常持器物**三者区分，不靠脸。
  吴月娘（正妻，端重，直身袄、佛珠）、李娇儿（二娘，丰腴淡漠，深绛缠枝褙子、
  描金妆匣）、孟玉楼／玩家（三娘，沉静，素缎、算盘与账册）、孙雪娥（四娘，简素，
  围裙、灶前布巾）、潘金莲（五娘，利落，窄袖、团扇）、李瓶儿（六娘，丰润，
  织金褙子、绣囊）。
- **庞春梅**不属六房：她是**通房丫鬟，始终无名分**，因此不进六房排行榜，也不共用
  六房的角色语法。她的视觉任务是**位置反转**——前期双鬟、捧茶盘、最纤细；
  第三幕改换**命妇冠服**（凤冠、织金大红礼服、霞帔、双手端仪），体量与气场明显放大。
  这是全书最完整的一次翻身，必须一眼看出是同一个人、却已不是同一个身份。
- **仆役**：比主子小一号、深色短打、必持**灯笼或食盒**——他们是移动的情报单位，
  轮廓必须与主子截然不同。

### 世界密度随家道变化

同一张宅院图三幕三种状态，是全作最重要的视觉叙事：

| 幕 | 宅院状态 |
|---|---|
| 一·入门 | 器物齐整，灯火半数，花木初盛 |
| 二·争锋 | 最盛：满院灯火、张灯结彩、廊下堆着节礼与账箱 |
| 三·星散 | 灯灭大半、器物被搬空留下浅色印痕、封条、落叶、空廊 |

## 材质、光与颜色

| 元素 | 处理 | 功能 |
|---|---|---|
| 底材 | 米白宣纸，可见纤维与轻微霉斑水渍 | 全作统一年代感 |
| 线 | 墨线白描，粗细分层；可见木刻刻痕 | 主要造型手段 |
| 设色 | 淡彩套印，色块略偏离线稿（套色错位） | 版画真实感，统一全部生成资产 |
| **金** | 金箔感描边与数字 | **专属明账**：位分、体面、宠、排行榜 |
| **墨** | 纯黑小字与细框 | **专属暗账**：私房、人情、退路 |
| **朱** | 朱砂印记与红点 | **专属风声**：谋算暴露风险 |
| 灯火 | 暖黄光晕，只出现在有人活动的房间与廊道 | **功能光**：哪里亮＝哪里有人在动，是情报本身 |
| 第三幕 | 全局降饱和，金色褪成灰白 | 明账作废的视觉表达 |

## 界面美术

- **框架**：宣纸底 + 细墨边框 + 硬木格心，仿明代册页与账簿装帧。
- **排行榜（前两幕）**：右上角金色竖排木牌，六房名次自上而下，每回合翻动一次。
  它必须做得**显眼、好看、令人在意**——它是全作最大的诱饵。
- **暗账面板**：左下角一枚不起眼的墨色小账册标签，默认折叠，展开是竖排小楷账目。
  前两幕刻意做得朴素，与金色排行榜形成对比。
- **传闻卡**：仿明代帖子／名刺，标注来源仆役与可信度（高／中／低用朱点数量表示）；
  被证伪的卡盖一枚朱色「妄」字印。
- **字体**：宋体／楷体系，需覆盖人名（潘金莲、李瓶儿、庞春梅、孟玉楼、吴月娘、
  孙雪娥、西门庆、官哥儿、孝哥儿、李衙内、薛媒婆、冯妈妈、玳安、小玉、韩道国）、
  节令名与账目数字，且允许商业发布。竖排只用于榜牌与账册，正文横排保证可读。
- **零 emoji、零现代图标字体**：所有功能标识用汉字印章（探／结／谋／持／藏）。

## 三张签名游戏画面

### 一 · 初见：过门（节令 1）

宅院剖面全景，玩家的院子刚点上第一盏灯。前景是抬进仪门的嫁妆箱笼，账房先生
执笔待记。**玩家动作**：选择嫁妆「入公中」还是「留一半在自己名下」。
第一焦点是那口半开的箱子——金色的锭子与墨色的账册同时在画面里，
**开局第一帧就把两本账摆在同一个箱子里**。右上角排行榜此刻还是空的。

### 二 · 核心动作：中秋夜的廊下（第二幕）

满院灯火，堂屋席面正盛（可见座次），而玩家的视线被廊下三盏移动的灯笼牵走——
玳安从潘金莲院里出来，往李瓶儿院去。**玩家动作**：把最后 1 点行动投给「探」
（买下玳安这一趟的消息）还是「藏」（把办节礼剩的银子存进私房）。
金色排行榜正显示你落到第四，墨色暗账在左下角安静地亮了一下。
**只有这部作品会产生这张画面**：情报是一盏在走的灯，代价是排行榜上的名次。

### 三 · 高潮／结果：第79回（节令 19）

家主暴亡。同一张宅院剖面图，灯火在两秒内逐间熄灭；右上角的金色排行榜木牌
**逐块翻落、掉出画面、不再出现**；全局色彩褪向灰白。与此同时，左下角那枚一直
不起眼的墨色小账册标签开始放大，最终展开占满全屏——竖排小楷，逐条列出你的
私房、你手上的人情、你有几条退路。屏幕正中先浮出一行金字「历史最高位次：第一」，
随即褪灰，下面接一行墨字「于结局无所增益」。

## 生成资产的方向约束

- **统一风格锚点**（每张必带）：明代绣像插图／界画白描／淡彩套印，可见刻痕与
  套色错位，米白宣纸底；**着装完整端庄、无裸露、无暧昧**；不要现代影视造型、
  不要 Q 版、不要水墨写意、不要辉光、不要文字、不要数值条。
- **人物立绘**：单人全身或半身，平涂纯亮绿背景 #00B140（纯净无阴影无渐变），
  便于本地扣像；统一取景比例与站位高度；同一角色需产出三幕三套服制（初嫁／
  鼎盛／缟素）。
- **宅院剖面**：全幅铺满，**预留廊道与右上、左下的 UI 空区**，避免细节与排行榜、
  暗账面板打架；同一构图产出三幕三种密度（齐整／鼎盛／空落）。
- **经皮肤层进入**：每个资产对应一个视觉键，缺图回退灰盒占位，美术与玩法并行推进、
  随时替换，不改引擎。

## 原创与授权边界

依据公版古典原著与明代版画、建筑、服饰、器物的通用形制重新设计。不使用任何影视、
动画、漫画、游戏或现代插画版本的角色造型、场景设计与界面元素；不复用受保护的
角色形象、地图、UI 与素材。字体须允许商业发布。

## 仍需验证的视觉风险

- **剖面图的信息密度**：六房 + 花园 + 前厅在 1280×800 内是否还能看清仆役动线，
  可能需要减少同屏可见房间数或加大人物比例。实机截图确认。
- **金／墨对比是否足够**：暗账做得太朴素可能导致玩家完全忽略它，反转变成挫败而非
  醒悟。需在试玩中确认玩家至少展开过一次暗账面板。
- **三幕三套服制的资产量**：七位主子（六房 + 春梅）× 三幕并不都必要。实际取舍是：
  常服全员必做；**缟素只做第三幕仍在场的人**（孟玉楼、潘金莲、吴月娘、孙雪娥）——
  李瓶儿第62回已亡、西门庆第79回已死，都不需要缟素；春梅第三幕不穿孝，
  改出**命妇冠服**。初嫁服制不单独出图，用常服代替。
