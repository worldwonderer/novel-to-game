# 游戏设计：《西游记：三借芭蕉扇》

## 受众、语言与文化边界

首发面向中文玩家，界面使用简体中文；玩家不需要读过原著，但原作术语、人物关系和
佛道民间信仰的区别不能被通用奇幻词替换。首次出现的生僻名物用战斗后果和一句短提示
解释，不用知识问答考玩家。策划保留一份简繁对应和专名术语表，为后续英文等本地化提供
依据；第一版不实现语言切换。

海外本地化必须重新验证名称、宗教概念、幽默和角色关系，不能只替换字符串。回合制指令、
五行相克、法术法力和携宠阵型的心智可以跨文化复用，《梦幻西游》《问道》一代玩家熟悉的
回合制西游只作为这套语法的证据，不作为西游文化和视觉身份的来源。

## 玩家承诺

你是孙悟空，带着一支五行互补的取经队伍走进火焰山这一段连续劫难。每回合你为每位伙伴
下达指令，看着双方按速度快慢逐个出手、飘出伤害数字与「克!」「被克」，再用五行、变化、
法宝和携宠把一场看似只能硬拼的对阵扭成可解的棋局。最终奖励不是通关动画，而是罗刹女的
真扇一息火、二生风、三落雨，师徒在雨后穿过八百里火焰山。

## 类型与体验支柱

这是《梦幻西游》《问道》风格的**回合制指令 RPG**：敌左上、我右下的经典对阵，顶部速度
行动顺序条，指令菜单出招，五行相克与法力法术构成战斗骨架。一段序幕小世界（点击移动、
问土地、找罗刹女）串起第五十九至六十一回的九段可玩桥段与结局，是「三调芭蕉扇」约 45-90 分钟的完整战役。

三个支柱按优先级排序：

1. **五行相克成体系**：金克木、木克土、土克水、水克火、火克金；克制 ×1.5、被克 ×0.66。
   选对属性、用对携宠与变化，是比堆伤害更重要的胜负手。
2. **指令回合可复现**：命中、暴击、伤害浮动全部走同一个种子随机（seedRNG）；同 seed
   同指令必得同结果，既保留回合制西游的手感，又让自测逐字节对齐。
3. **胜利写回世界**：假扇越扇火越旺、真扇三段息火生风落雨、牛魔王现出白牛真身、还扇后
   师徒西行，都是打赢之后世界真的改变，而非只弹「敌人已击败」。

发生范围冲突时，先删过渡演出、敌人招式和养成深度，不能删五行相克、真假扇三段因果和
牛魔王多阶段变身。

## 核心循环

```text
为每位伙伴下指令（攻击/法术/防御/道具/特技/自动/逃跑）
  -> 生成含敌方的行动队列，按速度从高到低排序
  -> 逐个结算：命中→暴击→五行系数→伤害浮动，屏幕飘字
  -> 每次行动后判定死亡与胜负
  -> 一方全灭则本场结束，否则进入下一回合
  -> 胜利后全员升一级、按等级解锁新法术，进入下一场
```

玩家的乐趣来自：读懂对阵里的五行关系，为当前敌人挑选克制它的角色、携宠、变化或法宝；
在法力有限时权衡攻击、法术、防御回蓝与道具；在真假扇、多阶段 Boss 这类剧情节点上做出
正确决策。

## 回合结构

一个回合严格按以下顺序推进，引擎（`js/engine.js` 的 `executeRound`）与界面共用同一套逻辑：

1. **我方逐个下指令**：界面让每位在场伙伴依次选择指令；玩家未指定的伙伴按「防御」补齐，
   敌方一律由 AI 补齐指令。
2. **生成行动队列**：把所有存活单位（含敌方）按**有效速度降序**排成一条队列（`buildActionQueue`）。
   速度已计入阵型、变化形态与增减益修正；同速时我方优先，再按单位 id 升序，完全确定。
3. **逐个结算**：沿队列取出每个单位——先清除其上回合的防御姿态、结算「落雨」持续回血、
   判定眩晕（被晕则跳过），再执行它这回合的指令并即时飘字。
4. **死亡与胜负判定**：每次行动后立即检查；有单位体力降到 0 则死亡（Boss 若有下一阶段则
   换阶段而非死亡）。敌方全灭 → 我方胜；我方全灭 → 战斗失败；胜负一旦判定立即结束本回合。
5. **回合末计时**：所有增减益回合数减一，到期移除；变化形态计时减一，到期还原五行。
6. **下一回合**：回合数加一，回到第 1 步。

没有站位格、没有移动步数、没有潜入与身份怀疑；一切在这条速度队列上以指令和数值结算。

## 战斗系统

### 属性

每个单位有六项属性，我方随等级成长，敌方随战役进度分级（战斗定义的 `enemyLevel` 逐幕上调，
使中后期仍有挑战）（`unitLevelStats`：`base + growth×(level-1)` + 潜力点加成）：

| 属性 | 作用 |
|---|---|
| HP 体力 | 归零即倒下（Boss 触发换阶段除外） |
| MP 法力 | 施放法术与七十二变的消耗，防御可回蓝 |
| 攻击 atk | 物理指令（`kind: phy`）的伤害基数 |
| 防御 def | 减免所受伤害；公式里按半值抵扣 |
| 速度 spd | 决定行动队列先后 |
| 灵力 mag | 法术指令（`kind: mag`）的伤害基数 |
| 五行 element | 金/木/水/火/土之一，决定相克关系；变化会临时改写 |

### 五行相克

相克链为**金克木、木克土、土克水、水克火、火克金**（`data.js` 的 `ELEMENTS`）。
攻方克防方，伤害 **×1.5** 并飘「克!」；攻方被防方克，**×0.66** 并标「被克」；无关系 ×1.0。
典型关系：悟空（金）克罗刹女（木）；辟水金睛兽（水）克一切火妖；八戒（木）克白牛真身（土）。

### 伤害公式（含五行系数，全程 seedRNG 可复现）

`calcDamage` 按固定顺序计算，每一步的随机都取自 `state.rng`（mulberry32 种子随机）：

```text
atkStat = 法术取灵力(mag)，物理取攻击(atk)      // 均为计入阵型/变化/增益后的有效值
base    = atkStat × 技能倍率 − 防御 × 0.5       // 下限 1
dmg     = base × 五行系数(1.5 / 0.66 / 1.0)
若暴击(概率 = 单位暴击率 + 技能暴击加成)：dmg × [1.5, 2.0) 随机
dmg × [0.9, 1.1) 随机浮动
若目标处于防御姿态：dmg × 0.5
dmg × (1 − 减伤增益)                            // 如沙僧罗汉金身
若目标为我方且阵型有受伤修正：dmg × 受伤系数     // 如六丁阵 0.9
最终伤害 = max(1, 四舍五入)
```

**命中**：`hitChance = (技能命中 ?? 0.95) + 命中增益`，钳制在 [0.05, 1]；未命中飘「Miss」。
基础攻击命中 0.95、倍率 1.0、不耗蓝。因为命中、暴击、暴击倍率、伤害浮动都走同一个种子，
`?seed=42` 固定后同一串指令必得同一结果，`test/battle.mjs` 据此逐字节断言。

### 指令

选中伙伴后从以下指令出招（`execCommand`）：

| 指令 | 效果 |
|---|---|
| 攻击 | 基础物理单体，命中 0.95、倍率 1.0、不耗蓝 |
| 法术 | 消耗 MP 施放已习得技能（单体/群体/增益/减益）；蓝不够或未习得则回退为普攻 |
| 防御 | 本回合受伤 ×0.5（持续到下次行动前），并回复约 15% 最大法力 |
| 道具 | 使用金疮药 / 法力丹 / 芭蕉扇（假扇或真扇） |
| 特技 | 悟空「七十二变」，或「切换阵型」（消耗该单位本回合行动） |
| 自动 | 交由 AI 为该单位代打一回合（与敌方 AI 同一套 `aiCommand`） |
| 逃跑 | 非 Boss 战 70% 成功、直接结束本场返回小世界重试；Boss 战必定失败 |

## 技能与法术

全部技能定义在 `data.js` 的 `SKILLS`，`kind: phy` 用攻击、`kind: mag` 用灵力，按等级解锁：

### 孙悟空（金，暴击 15%，可七十二变）

- **Lv1 如意金箍棒**（物理·单体，倍率 1.8，耗蓝 6，命中 0.95）：物理单体高伤。
- **Lv2 火眼金睛**（法术·单体，耗蓝 12）：识破破绽，使敌防御 −35%（3 回合）。
- **Lv3 横扫千军**（物理·群体，倍率 1.15，耗蓝 10，命中 0.92）：物理群攻。

### 猪八戒（木，暴击 10%）

- **Lv1 九齿钉耙**（物理·群体，倍率 1.05，耗蓝 6，命中 0.92）：物理群攻。
- **Lv3 拱地**（物理·单体，倍率 0.8，耗蓝 10，命中 0.9）：65% 概率击晕敌一回合。

### 沙悟净（土，暴击 8%）

- **Lv1 降妖宝杖**（物理·单体，倍率 1.35，耗蓝 5，命中 0.97）：稳定单体，自愈 35% 伤害量。
- **Lv2 罗汉金身**（法术·全队增益，耗蓝 15）：全队受伤 −30%（3 回合）。

### 辟水金睛兽（水，暴击 10%，携宠）

- **Lv1 辟水诀**（法术·群体，倍率 1.25，耗蓝 10）：水系群攻，克火。
- **Lv2 金睛**（法术·全队增益，耗蓝 8）：全队命中 +15%（3 回合）。

## 七十二变（悟空特技）

「特技 → 七十二变」消耗 **10 点法力**，让悟空在 **3 回合**内临时改写五行、属性并替换技能栏
（`FORMS`）。四种形态：

| 形态 | 五行 | 属性修正 | 变化技 | 定位 |
|---|---|---|---|---|
| 金甲神将 | 金 | 攻 ×1.4 | 神将横扫（物理群体，倍率 1.3） | 群攻输出 |
| 赤焰灵猿 | 火 | 灵力 ×1.45 | 烈焰拳（火系单体，倍率 1.7） | 高爆法术，克金 |
| 玄甲龟将 | 水 | 防 ×1.6、速 ×0.8 | 玄冰击（水系单体，倍率 1.7） | 抗压兼克火 |
| 蟭蟟虫 | 金 | 速 ×1.5、攻 ×0.8 | 无 | 抢速自保 |

变化不是伤害皮肤：改五行意味着改相克关系（如赤焰灵猿以火克敌方的金），玄甲龟将换来
防御却牺牲速度，蟭蟟虫抢到先手却几乎不输出。形态在回合末计时，到期自动还原本来五行。

**化虫入腹彩蛋**：战斗一（罗刹女）设有 `transformFinisher`，当罗刹女体力 ≤ 55% 时悟空施展
七十二变，即触发「化虫入腹」直接取胜——对应原著第 59 回悟空钻进罗刹女腹中逼其交扇。这是
教学彩蛋，奖励玩家在正确时机使用变化，而非又一次硬拼。

## 携宠协战

辟水金睛兽（水）从战斗二起作为第四名成员入队（`petJoined`）。它本是牛魔王坐骑，水系身份
让它克制火焰山所有火妖：辟水诀是唯一的水系群攻，金睛为全队补命中。战斗一悟空、八戒、
沙僧三人上阵；战斗二、三携宠四人同场。携宠对应原著悟空在碧波潭偷走辟水金睛兽的情节。

## 阵型

`FORMATIONS` 提供两套阵型修正，作用于全体我方：

- **天罡阵**：攻击 +12%、速度 +5%——抢先手、拼输出。
- **六丁阵**：防御 +18%、受伤 −10%、速度 −5%——扛 Boss 高伤时保命。

战斗外在顶栏「阵型」切换；战斗内用「特技 → 切换阵型」，会消耗该单位本回合行动，因此换阵
本身是一个需要权衡的战术选择。

## 芭蕉扇

芭蕉扇是道具，不是装备；它的分量在于改写整场对阵的状态，且真假两段完全对立：

### 假扇（芭蕉扇·伪）——反噬

战斗二拿到的假扇一经使用，场上所有**火敌攻击 +30%**（3 回合）——越扇火越旺。这照搬原著
第 59 回「一扇火更旺」的骗局：玩家若照百姓说法猛扇，反而助长火势。它教会玩家法宝要验真伪，
真正的翻盘要靠真扇。

### 真扇（芭蕉扇）——三段息火生风落雨

BOSS 战携带真扇 ×3，按次数递进（`fanStage` 累计封顶 3），是牛魔王战的翻盘关键：

| 次序 | 效果 | 原著对应 |
|---|---|---|
| 一扇·息火 | 清除敌方增益（保留我方所加的破防/眩晕），火敌攻击 −30%（3 回合） | 一扇息火 |
| 二扇·生风 | 全队速度 +30%（3 回合），抢下整条行动队列的先手 | 二扇生风 |
| 三扇·落雨 | 全队持续回血（每次行动回 8% 最大体力，3 回合），火敌防御 −25% | 三扇下雨 |

三段既是战力曲线，也是「三借芭蕉扇」的仪式：先压制火敌、再抢速、后续航破防，配合携宠与
变化拿下白牛真身。

## 敌人与多阶段 Boss

敌人定义在 `ENEMIES`，招式取自 `SKILLS`，AI 走统一的 `aiCommand`（辅助优先、群敌用群攻、
否则单体或普攻）：

| 敌人 | 五行 | HP | 招式 |
|---|---|---|---|
| 罗刹女 | 木 | 980 | 双剑击（物理单体 1.35）、芭蕉扇风（物理群体 0.85） |
| 火兵 | 火 | 430 | 火弹（火系单体 1.25） |
| 火炎校尉 | 火 | 640 | 火弹、烈焰喷吐（火系群体 1.0） |
| 牛魔王 | 火 | 1500 | 混铁棍（物理单体 1.55）、蛮牛冲锋（物理群体 1.15） |
| 白牛真身 | 土 | 1950 | 法天象地（物理单体 1.9）、地火燎原（火系群体 1.2）、蛮牛冲锋 |

**多阶段 Boss**：牛魔王数据带 `nextPhase: whitebull`。当牛魔王（火）体力被打空，不会死亡，
而是**现出白牛真身**（土）——满血重开、清空增益与变化，换立绘、换五行、换更强招式与属性
（`checkDeath` 触发 `phase` 事件）。这正是原著第 61 回牛魔王现出大白牛原形的一战：五行从
火转土后，克制关系随之改变（八戒木克白牛土成为破局点），迫使玩家重新组织真扇三段、携宠
和变化。

## 战役流程

一段序幕小世界 + 第五十九至六十一回的九段可玩桥段 + 结局（`main.js`），约 45-90 分钟。完整顺序：
序幕·火焰山脚 → 罗刹女一战（罗刹女+芭蕉洞侍婢，一扇吹飞悟空）→ 灵吉菩萨授定风丹 → 再战·化虫入腹取假扇
→ 火焰山·假扇反火（火兵，校尉召唤波次）→ 摩云洞·玉面公主（+妖将）→ 初战牛魔王（赴宴而走）→ 碧波潭·变螃蟹
偷辟水金睛兽（获召唤兽）→ 骗真扇 → 牛魔王变假八戒反骗 → 积雷山决战（牛魔王+玉面公主+妖将群，八戒/土地接力）
→ 众神围剿（哪吒/李天王剧情门控支援）→ 白牛真身（多阶段+狂暴，真扇落雨破绽期集火）→ 真扇三段+四十九扇
→ 还扇西行结局。下列列出其中几处关键战斗：

### 序幕·火焰山脚（小世界）

点击移动，向土地问话得知火焰山与铁扇公主的来历，走到罗刹女处触发战斗一。开局队伍全 1 级，
持有金疮药 ×2、法力丹 ×1，天罡阵。

### 一、翠云山·芭蕉洞外（战斗一）

对阵罗刹女（木），悟空、八戒、沙僧三人上阵，带新手教学。悟空（金）克罗刹女（木）；把她
压到 55% 体力以下再七十二变可触发化虫入腹速胜。非 Boss 战可 70% 逃跑回小世界重试。
胜利后追加假扇 ×1 与金疮药 ×1，进入火焰山。

### 二、火焰山·火口（战斗二）

对阵火兵·甲、火兵·乙、火炎校尉（皆火），辟水金睛兽入队携宠四人上阵。辟水诀（水）克全场火敌；
此处玩家会见识假扇的反噬。胜利后交出假扇，换得真扇 ×3。

### 三、积雷山·摩云洞前（BOSS 战）

对阵牛魔王，两阶段（牛魔王火 → 白牛真身土），携宠四人，携真扇 ×3，**不可逃跑**。用真扇
三段息火生风落雨、以八戒木克白牛土、配合携宠与变化降伏白牛真身。

### 结局·还扇西行

## 升级养成

围绕战役展开、决定论可复现的成长（不做门派、抽卡、装备打造/交易、技能树）。所有养成数据都是
`makeUnit`/`unitLevelStats`/`createBattle` 的**可选入参、缺省逐字节回退**，升级与捕捉的随机走**独立
campaign 种子**、绝不碰战斗 `state.rng`，因此不破坏引擎的可复现断言：

- **升级加点（潜力点）**：每胜升一级并发潜力点，玩家在「角色」面板自由投 体/攻/防/速/灵（含「推荐加点」
  一键，走独立种子可复现）；设软台阶防线性爆炸。
- **法术熟练**：已习得法术随等级熟练进阶（威力 +6%/级、耗蓝 -2，上限 3）；配合 MP 经济让后期多放一次技能。
- **按等级解锁法术**：技能表以等级为键，解锁扩展到 Lv6（悟空/八戒/沙僧各有中后期新技），升级不再纯数值空转。
- **武器/防具装备（1 件/角色）**：剧情掉落的简单 flat 属性装备（如如意金箍棒·精 攻+12），在 `makeUnit`
  阶段叠加；不做打造、词条与交易。
- **规则型法宝（1 件/全队，忠原著）**：只做规则效果不加攻——定风丹（速+6、免疫扇风减速，灵吉菩萨赠·第59回）、
  避火锦（火系受伤 -25%）二选一；守 SOURCE_BIBLE「不把法宝降格为伤害更高的装备」。
- **召唤兽/宝宝**：辟水金睛兽（碧波潭变螃蟹偷得，水克火）与火妖赤焰火骝（火焰山用捕妖绳、血气≤40% 剧情门控
  捕捉）为固定协战单位；「召唤兽」面板选 1 只上阵、占正式行动位、吃 MP 与成长；不做收集池、资质、繁殖与交易。
- **物品/背包**：丹药（回体/回蓝）、捕妖绳（捕召唤兽）、芭蕉扇（真/假，规则型）等；固定剧情节点获取，不刷不卖。
- **不做的养成**：无抽卡、无门派数值墙、无装备打造/交易、无 PVP、无 MMO 长线；成长服务这段战役的节奏，不做无限刷。

## 失败与重试

- **败北**：我方全灭即战斗失败；战役包装层（`runBattle`）换一个种子偏移自动重开当前战斗，
  凭养成进度与更优指令再战，不回退整局。
- **逃跑**：仅非 Boss 战可用，70% 成功后回到小世界重试；Boss 战禁用逃跑。
- **无死局**：三场战斗均可凭现有队伍、道具与阵型打通；真扇三段、化虫入腹、多阶段变身都是
  设计内的可用手段而非陷阱。

## 存读

- 顶栏「存档 / 读档」写入 `localStorage`；**战斗中不可存读**。
- 存档记录战役阶段、四名单位等级、携宠状态、阵型、道具与种子基数，**不记录战斗中途状态**。
- 每场胜利后自动存档；标题画面有存档时提供「继续」。
