# 美术方向：《西游记：三借芭蕉扇》

## 视觉定位

做一部**明代彩印神怪 · 回合制西游战斗**：固定单屏战斗舞台上，全屏是一张多色木刻
风景，队伍与妖魔像戏台上分列两厢的角儿，下方压着一条宣纸卷轴木框的指令台。整体
质感来自明代绣像版画与木刻套印——套色错位的边线、有限的矿物色、朱与墨的骨架——
但它是一场可以逐条下令、按速度结算、看五行相克的**指令回合制 RPG**，不是棋盘，
也不是等距格子。

《梦幻西游》《问道》只提供“指令回合制在单屏里清楚可读、敌我对阵一眼能认”的形制
参照，不借它们的 Q 版大头、现代描边和塑料高光；《大神》只提供“通关后世界被修复”
的结果参照，不借水墨笔触；任何现代西游影视和游戏都不是角色美术参考。

文化依据来自公版百回本文字、明代绣像版画、传统戏台和寺观彩塑研究。佛、道、妖、
民间生活的服饰和符号分别核对，不混成通用“东方仙侠”。首发简体中文使用完整 CJK
字形、贴近木刻的宋体，技能名与提示保持现代玩家可读；文字不烙进立绘或背景图，全部
由界面层排字。

## 视觉支柱与反向支柱

1. **背景是一张会呼吸的木刻风景**：翠云、火焰、积雷各有独立套色与地平线；不铺通用
   仙侠辉光，也不让一张红黑火山、一张青绿洞府敷衍到底。战斗结算改变的是这张风景
   本身，而非盖在它上面的弹窗。
2. **立绘是戏台亮相，不是卡面**：每个单位保留戏曲亮相的重心与轮廓侧影，敌左上、
   我右下斜列相向而立；不做手牌、不做悬浮卡框、不让血条比人还大。
3. **变化先换形态轮廓**：悟空七十二变换的是整具身形与五行气色——金甲神将挺，赤焰
   灵猿窜，玄甲龟将沉，蟭蟟虫贴——不让不同形态共用一套立绘和一层描边。
4. **体量即信息**：牛魔王人形与白牛真身是两种体量，真身立绘明显放大、双角像移动
   山门，玩家一眼看出 BOSS 进入第二阶段；辟水金睛兽作为坐骑宠始终矮伏在队列里，
   不与主将抢轮廓。
5. **胜负写在世界上**：真扇三段让火焰山逐层退色——朱红火版落灰、青白风版清尘、
   深蓝雨版落在火源，最后转入雨后晴土；不以“敌人已击败”的字条代替环境改变。

避免通用仙侠辉光、红黑一色火山、青绿一色洞府、Q 版大头、现代影视脸谱、塑料描边
高光、随机跳动的巨大血条、漂浮任务标记，以及不能实际游玩的宣传海报式构图。

## 战斗镜头与舞台构图

单屏固定战斗舞台，不做自由旋转与漫游相机。全屏一张木刻背景，下方压一条约六分之一
屏高的指令卷轴；上缘一条半透明的行动顺序条。固定构图让“敌左上、我右下”“速度从
高到低”“五行相克”始终有稳定含义，也降低网页实现与遮挡风险。

- **经典对阵**：敌方沿左上斜列展开，我方沿右下斜列相向；我方立绘水平翻转朝左，
  与敌相视，形成清楚的两厢戏台关系。
- **焦点次序**：当前下令/行动的单位由头顶朱金指示与立绘外光提示，目标可选时立绘
  描金脉冲。相机不移出舞台，玩家在提交指令前始终看得到全场。
- **体量分级**：普通单位与放大的白牛真身共用同一舞台；真身占据更大画幅但不破坏
  斜列关系，用体量而非特效说明威胁升级。
- **结算克制**：受击轻抖、暴击与冲锋触发一次短促全屏震颤即收，随后回到静止构图；
  不叠加持续晃动与满屏粒子，保证连续下令时画面可读。

每张舞台都朝西留出出路的暗示：前几战被火墙、洞门或强敌封住，真扇落雨、火根断绝后，
西路才在雨后晴光里打开。

## 战斗舞台的世界语法

### 序幕 · 火焰山脚（overworld）

一张俯视点选的木刻小世界：赭红焦土、断续热浪地平线、远处芭蕉洞与铁扇仙居的轮廓。
玩家点地移动、点土地问话、寻罗刹女。它是一段问路调查而非战斗，用同一套宣纸提示条
与朱框对话框把“先查清、再动手”的西游循环立在开场。

### 翠云山 · 芭蕉洞外（cuiyun）

冷绿松竹、湿黑石阶、青白水沟，芭蕉洞门居中。罗刹女（木）持双青锋居左上，师徒与
悟空（金）居右下，金克木的相克关系写在开场对阵里。这里是化虫入腹彩蛋的舞台，冷绿
背景要能在体力见底时压出一道朱金高光。

### 火焰山 · 火口（huoyan）

朱红焦砖、炽白砖边、烟黑剥落构成人工火层，地平线是八卦炉余砖的断续火源，不是岩浆海。
火兵、火炎校尉（火）成群居左上；此战携辟水金睛兽（水）压阵，水克火的克制在背景色
与飘字里同时成立。假扇一旦出手，火层要能整体转旺、朝队伍逼近。

### 积雷山 · 摩云洞前（leiji）

灰紫天光、赭色岩台、缠绕的护法云气。牛魔王（火）人形持混铁棍居左上，混铁棍与蛮牛
冲锋是两种进攻侧影；转入白牛真身（土）后立绘放大、法天象地。这张舞台要能承接真扇
三段的世界重印：由灰紫压抑逐步转为雨后清明。

## 角色与形态语法

- **孙悟空（金）**：毛脸、雷公嘴、紧箍、虎皮裙与金箍棒是基础识别；精瘦、前倾、动作
  快收，不照搬现代影视冠甲与红金套装。他是唯一能七十二变的单位，底部指令台以「特技
  → 七十二变」标示身份。
- **猪八戒（木）、沙悟净（土）、唐僧**：八戒低宽轮廓配九齿钉耙；沙僧持降妖宝杖、
  轮廓稳重；唐僧不参战、随队剧情，出现在序幕与结局的对话立绘而非战斗队列。三者五行
  各异，立绘侧影即分工。
- **辟水金睛兽（水，宠）**：牛魔王坐骑，被悟空夺来协战。矮伏兽形、水色气质，站在
  队列里作辅助与克火，不与主将争高。
- **罗刹女（木）**：依据第 59 回的团花手帕、锦云袍、绣裙、弓鞋与双青锋；大弧扇面与
  细直双剑是两组进攻侧影，不使用现代固定造型。
- **牛魔王（火）→ 白牛真身（土）**：人形依据第 60 回的银亮铁盔、锦绣金甲、卷尖靴与
  混铁棍；真身是巨型白牛，立绘明显放大、双角如铁塔、肩背像移动山门。两形态是两条
  五行、两种体量，阶段转换靠换立绘与放大，不靠换色滤镜。
- **悟空变化**：金甲神将（金·攻）、赤焰灵猿（火·法）、玄甲龟将（水·防）、蟭蟟虫
  （金·抢速）各是一具独立形态立绘，附带各自的五行徽记与气色。变化时以木刻套印
  错位的三层短暂错开再套准成新形；只有当前单位的五行徽记与形态标签向玩家说明身份，
  世界内不留猴尾金光。

立绘落地由脚步、衣摆和地面接触阴影表达，不套统一圆盘底座，避免画面变成通用手游
战棋。

## 材质、光与颜色

界面基色沿用宣纸木刻一套：宣纸米 `#f2e8d5`、墨 `#2b2118`、朱 `#a8322a`、金线
`#c9a227`、暗木 `#4a3423`、水青 `#2a6b8a`。功能色不被氛围滤镜吞掉。

| 功能 | 颜色与材质 | 形状/辨识辅助 |
|---|---|---|
| 五行 · 金 | 土黄铜 `#b8912a` 圆徽 | 立绘左上圆形五行徽记 + 「金」字 |
| 五行 · 木 | 松绿 `#3e7d3a` 圆徽 | 圆徽 + 「木」字 |
| 五行 · 水 | 水青 `#2a6b8a` 圆徽 | 圆徽 + 「水」字 |
| 五行 · 火 | 焦朱 `#b03a24` 圆徽 | 圆徽 + 「火」字 |
| 五行 · 土 | 赭褐 `#7d5a34` 圆徽 | 圆徽 + 「土」字 |
| 体力 HP | 朱红渐变条 | 立绘下窄条 + 数字 |
| 法力 MP | 水蓝渐变条 | 立绘下窄条 + 数字 |
| 增益/减益 | 暗木底金线小签 | 立绘下横排状态签，附文字 |
| 火与热 | 朱红、炽白砖边、烟黑剥落 | 向外鼓动、假扇后整体转旺 |

火焰山用低角度热反射，翠云山用树隙冷光，积雷山用灰紫天光、雨后转均匀晴光，序幕小
世界用赭红焦土日光。五行徽记与飘字的功能色在任何背景下都保持恒定，做到色盲可读。

## 界面美术

底部指令台是一整块宣纸卷轴木框：暗木粗边、内嵌一道内缩金线（`outline` 内移约
6px），左侧一格状态条报当前单位与提示，右侧四列指令按钮阵。按钮为木刻朱印质感，
按下有两像素下沉与压痕，禁用态降饱和灰化。

- **指令菜单**：攻击 ⚔ / 法术 ✦ / 防御 ⛨ / 道具 ❖ / 特技 ✪ / 自动 ↻ / 逃跑 ➾，
  每命令附独立木刻符号，进入子项（法术表、道具表、七十二变、切换阵型）时在同一台面
  内展开，不弹多层浮窗、不用技能盘。
- **行动顺序条**：舞台上缘一条半透明暗木条，按速度从高到低排列各单位小头像签，
  当前行动者描金高亮、已行动者压暗，让“先谁后谁”一眼可数。
- **HP/MP 与状态**：每个立绘下方一组窄条——朱红体力、水蓝法力，条内叠数字；其下
  一排状态小签用文字写明增减益，不靠纯图标。
- **飘字伤害**：普通伤害米黄，暴击更大更红（约 32px 大红上跳）；克制打出金色更大的
  「克!」，被克打出更小的灰字「被克」；治疗为绿字、回蓝为蓝字、未命中为灰字、增益
  为淡蓝。数值差异同时由颜色、字号与文字标签三重冗余表达，色盲可辨。
- **通用框**：标题、教学、阵型选择、升级结算、暂停与结局共用同一套暗木＋金线宣纸框，
  与战斗台面同源；键盘焦点、色盲形状与桌面视口内的文字边界必须可见。

## 反馈与结算

- **变化**：七十二变换形时立绘做一次木刻套印错位再套准，新形态带绿意外光与新的五行
  徽记；界面立即刷新该单位的五行、属性与可用技能，让玩家看懂“换了打法”，而非只换皮。
- **化虫入腹**（第 59 回彩蛋）：战斗 1 中罗刹女体力见底、悟空变化后触发——一只木刻
  小虫自画面外飞入，压暗全屏、落定入腹，随即以一段旁白结算直接取胜。这是一张“智取
  胜于蛮打”的签名结算，不做血腥表现。
- **假扇反噬**：假扇出手时先抽走环境声势，再让火层整体转旺、火敌攻势抬升，用世界的
  反应否定玩家预期，而不是弹一条错误提示。
- **真扇三段 · 世界重印**：真扇是 BOSS 战翻盘关键，逐段改写火焰山这张风景——
  一息火：清敌增益、火敌减攻，朱红火层落为骨白余热；
  二生风：全队加速，青白风纹扫过、清去浮灰；
  三落雨：全队持续回血、火敌破防，深蓝雨点落在余火之上。
  三段各配一条对应色的技能横幅（真扇段用青绿描边），结算后画面转入雨后的洁净晴土，
  为结局“还扇西行”的雨幕做底。
- **BOSS 阶段转换**：牛魔王倒下时立绘不消失而是撑起白牛真身，体量放大、五行由火转
  土，一条阶段横幅点明“真身现”，让升级的威胁被看见而非被告知。
- **节奏**：一次世界重印或大结算后留半拍安静，再显示下一回合行动顺序，避免信息同时
  爆发。

## 声音方向（后续）

当前切片未接声音；美术方向预留其形制：以戏台锣鼓与寺观法器为骨——金箍棒落点用
短木铁撞、九齿钉耙用宽短土声、变化用木鱼板鼓加一声气流的三拍、真扇三段依次抽走
燃烧低鸣、加入风中衣料、再加入雨点与百姓人声。声音只从传统戏曲与法器方向设计，
不取用任何现代影视配乐、配音与音效素材。

## 五张签名战斗画面

### 序幕 · 火焰山脚问土地

俯视木刻小世界里，悟空站在赭红焦土上，远处热浪地平线与芭蕉洞轮廓被封在西路尽头。
土地头顶一枚朱印对话标记，宣纸对话框自下浮起报出铁扇仙来历。画面在动手之前——
先问、先查，把西游循环立在开场，而不是直接进战斗。

### 翠云山 · 化虫入腹

冷绿松竹舞台上，罗刹女（木徽）在左上、体力条见底泛红；右下悟空刚完成七十二变，
一只木刻小虫自画面外划入、全屏压暗。金克木的徽记对位清楚，「克!」金字尚在半空。
这是“智取取胜”的定格，不是一记重击的定格。

### 火焰山 · 携宠克火

朱红焦砖火口，火兵与火炎校尉（火徽）成群列于左上；右下队列里辟水金睛兽（水徽）
抬手放辟水诀，一片青蓝水纹压过成排火敌，每个头顶跳出金色「克!」。背景的朱与技能的
蓝在同一屏里说明“水克火”，玩家一眼读懂这场为何要带宠。

### 积雷山 · 白牛真身 · 真扇落雨

牛魔王已撑起白牛真身，放大的立绘占据左上、双角如铁塔、法天象地压来；右下悟空展开
真扇进入第三段，深蓝雨点垂直落在余火之上，火敌破防、全队浮起绿色回血字。灰紫天光
正被雨幕清开，是“真身现”与“三扇成雨”叠在一格的翻盘定格。

### 结局 · 还扇西行

雨后晴土，火焰山的朱红火层已退成洁净坡地，均匀雨幕转为薄光。师徒与白马沿打开的
西路远行，悟空把真扇归还罗刹女。宣纸卷轴框内落下结局题字——胜利写在被修复的世界
与继续向西的背影上，而不是一句“通关”。

## 原创与验证风险

事实与角色衣着只依据公版原著。明代绣像版画、传统戏台与寺观彩塑只提供公共领域的
轮廓、套色与材质研究方向；不得使用现代影视、动画、游戏截图、模型、音乐、配音与
角色设计作为直接素材。《梦幻西游》《问道》仅作“指令回合制单屏可读”的形制参照，
不借其画风。

实机必须验证：敌左上、我右下的对阵在满屏立绘下是否始终可辨；五行徽记、HP/MP 条与
克制飘字在色盲模式下是否可区分；七十二变四形态是否各有独立轮廓而非换色；牛魔王
→ 白牛真身的体量升级是否一眼可读；真扇三段息火·生风·落雨后火焰山这张风景是否
真的逐层改变；所有五张签名画面都能从实际战斗状态自然到达。
