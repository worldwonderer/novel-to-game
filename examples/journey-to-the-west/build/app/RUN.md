# 西游记 · 三借芭蕉扇(回合制指令 RPG)

《梦幻西游》《问道》风格的回合制指令 RPG:指令回合制、敌左上我右下经典对阵、
速度行动顺序条、五行相克、法术/MP、携宠协战、阵型、悟空七十二变、种子随机可复现。
零构建,原生 ES Module + Canvas/DOM。

## 启动

```bash
python3 -m http.server 5173
# 浏览器打开 http://127.0.0.1:5173/
```

目标视口 ≥1280×800,鼠标优先(键盘 Esc 可取消选目标)。

URL 参数(可选):`?seed=42` 固定全局种子;`&fast=1` 加速动画(自测用)。

## 玩法

- **流程(第59-61回全战役)**:标题 → 序幕·火焰山脚 → 翠云山一战罗刹女(+侍婢,第3回合被吹飞·剧情)
  → 灵吉授定风丹 → 再战取假扇 → 火焰山火兵战(假扇反火;校尉召唤;捕妖绳收服赤焰火骝)
  → 摩云洞·玉面公主对峙(+妖将×2)→ 初战牛魔王(赴宴而走·剧情)→ 碧波潭·变螃蟹偷金睛兽
  (三节点:变化/潜入/偷兽,选错退回重选)→ 辟水金睛兽入队 → 变牛魔王骗真扇
  → 假八戒反骗(可识破) → 积雷山决战(牛魔王+玉面公主+妖将 多人;八戒土地接力回血;
  白牛真身;哪吒率众神门控助战;真扇三段)→ 四十九扇断火根(结局演出)→ 还扇西行。
- **回合**:每回合先为每位伙伴依次下指令(攻击/法术/防御/道具/特技/自动/逃跑),
  下完后双方按**速度从高到低**依次行动(顶部行动顺序条,带速度值与「抢」位标记),逐个结算并飘字。
- **五行相克**:金→木→土→水→火→金(战斗左下角常驻相克环)。克制 ×1.5 飘「克!」印章,
  被克 ×0.66 标「被克」;悬停目标可预览 克/被克/普通。悟空(金)克罗刹女(木);
  辟水金睛兽(水)克火妖;八戒(木)克白牛真身(土)。
- **MP 经济**:法力金贵——一场 BOSS 战每人只够放 4–5 次技能;普攻穿插防御(回 5% 蓝),
  「何时开大」是核心决策。
- **敌方 AI**:敌方单体技锁定我方当前血量最低者;BOSS 每 3 回合蓄力预警、
  下回合重击(约 60% 最大血)——让血量最低者防御,或提前开罗汉金身。BOSS 战「自动」只普攻。
- **指令**:攻击不耗蓝;法术耗 MP(单体/群体/增益减益,按钮标注 目标·类型·五行·MP);
  防御减伤一半并回蓝;道具(金疮药/法力丹/芭蕉扇);特技=悟空「七十二变」;
  自动=AI 代打一回合;非 BOSS 战可逃跑(70%)。
- **变化**:无消耗,三回合内改变五行/属性/技能:金甲神将(金·群体)、赤焰灵猿(火·火法)、
  玄甲龟将(水·硬克火敌、被克伤害减半)、蟭蟟虫(金·抢速);形态按钮会标注能克到场上谁。
  战斗1中罗刹女体力≤55%时变化,可触发「化虫入腹」直接取胜。
- **阵型**:天罡阵(攻+20% 速+5% 但受伤+15%)/ 六丁阵(防+18% 受伤×0.8 速-5%)。
  战斗内点右上「阵型」按钮**免费切换,每回合一次**;战斗外用顶栏「阵型」。
- **连击**:暴击时 25% 概率追加一次减伤普攻,飘「连击!」。
- **芭蕉扇**:假扇使火敌攻击提升(反噬);真扇三段**无视敌方属性**:一息火(清敌增益+全体减攻30%)、
  二生风(全队提速)、三落雨(持续回血+敌方破防25%+破绽:受伤+40%)。白牛真身每回合狂暴叠攻8%,
  开落雨破绽集火速决。
- **成长**:每场胜利全员升一级 + 每人 5 潜力点(顶栏「角色」面板投 体/攻/防/速/灵,或一键推荐加点);
  法术熟练随等级自动进阶(威力+6%/级、耗蓝-2,上限3);技能解锁至 Lv6。
- **装备/法宝**:武器小幅 flat 属性(剧情掉落);法宝只做规则型(定风丹:速+6 免疫减速;避火锦:火伤减免25%),不加攻。
- **召唤兽**:辟水金睛兽(碧波潭三节点偷得)+ 火焰山用「捕妖绳」收服血气≤40% 的火妖(赤焰火骝);
  顶栏「召唤兽」面板选 1 只上阵(占正式行动位),宠同样吃成长/加点/熟练。
- **剧情演出**:一扇吹飞/赴宴而走(剧情退出,非失败);假八戒反骗(识破与否影响开局);众神围剿
  (白牛≤50%哪吒一次性助战,门控);四十九扇断火根(结局演出,非刷怪);还扇西行(不杀罗刹女/牛魔王)。
- **存读**:顶栏「存档/读档」(localStorage),战斗中不可用;每场胜利后自动存档。
- **音频**:Web Audio 程序化合成(零素材):五声音阶 BGM 分场景(标题/小世界/战斗/BOSS/结局),
  全套战斗音效(命中/暴击/克!/变化三拍/真扇三段/蓄力/胜负);顶栏「音效/静音」开关,状态持久化。
- **界面**:明代木刻/卷轴风——多层木框+金线+角饰+微浮雕木牌按钮;汉字金框徽(攻/法/防/道/技/自/逃、
  金木水火土、体攻防速灵),零 emoji 零图标字体;顶栏暗木卷轴条(角色/背包/召唤兽/阵型|存档/读档/音效/帮助,
  当前面板按钮朱底高亮);战斗卷轴指令台(当前单位头像牌);HP/MP 竹节分段条(掉血残影、低血脉冲);
  统一卷轴面板系统(背包图标格/召唤兽上阵与加点/角色五维加点/阵型/帮助)。

## 自测

```bash
node test/battle.mjs        # 引擎:固定 seed 可复现(146 项断言)
python3 test/qa_browser.py  # 浏览器全程:0 控制台报错,截图 → /tmp/xiyou_shots/
```

- `test/battle.mjs`:node 直接跑纯引擎。断言五行相克系数、伤害公式与暴击序列
  同 seed 逐字节一致、行动顺序速度降序、阵型修正、全战役六场战斗胜负、
  化虫入腹彩蛋、MP 消耗与防御回蓝、假扇反噬、真扇三段、BOSS 阶段转换、
  逃跑规则、升级解锁、成长加点与召唤兽收服。
- `test/qa_browser.py`:playwright 走通 标题→序幕→全战役(六场战斗)→结局→读档,
  覆盖教学、法术/特技/道具菜单、目标选择、变化、阵型切换(顶栏+战斗内)、
  假扇、真扇三段、白牛真身、成长面板、双召唤兽、存档/读档;
  断言控制台 0 报错(64 项浏览器断言)并保存约 47 张截图。
  自带静态服务(若 5173 已被占用则复用);`QA_SLOW=1` 可看常速动画。

```
  assets.js           资产加载;缺图回退色块头像/渐变背景
  ui.js               对话框/模态/飘字/提示条/顶栏
  audio.js            Web Audio 程序化音频:五声 BGM 分场景循环 + 合成 SFX,静音持久化

  qa_browser.py       playwright 全程自检

```

## 扩展点

- **新角色/敌人/技能/阵型/道具**:只改 `js/data.js` 数据表,引擎与界面零改动。
- **新战斗**:在 `data.js` 的 `BATTLES` 加一项(敌人列表、背景、是否 BOSS、彩蛋),
  在 `main.js` 战役流里接一个 `runBattle(...)`。
- **新形态**:`data.js` 的 `FORMS` 加一项(五行/属性修正/附带技能)即可进「七十二变」菜单。
- **BOSS 多阶段**:敌人数据 `nextPhase` 指向下一个形态定义。
- **缺美术**:`assets.js` 按键名加载,缺失自动回退色块头像与渐变背景,不影响运行。

## 已知边界

- 存档记录战役阶段与养成,不记录战斗中途状态;战斗中不可存读。
- 逃跑仅战斗1可用(回小世界重试);败北后换种子重试当前战斗。
- 目标视口桌面 ≥1280×800;未做移动端适配(音频已内置,见上「音频」)。
