# 成品目标

构建《西游记：三借芭蕉扇》：一段 18-22 分钟、桌面浏览器可玩的单人 2.5D 等距回合制
潜入策略游戏。玩家预读火势、视线和敌方意图，用变化改变走法与身份权限，用推位和
八戒接力围困牛魔王，最后以真扇永久改写火焰山棋盘。

第一版的质量标志是规则可读、结果确定、五张棋盘形成完整因果，不是角色数量、写实
资产或战斗特效。

首发界面只实现简体中文。所有玩家可见文案集中管理，不把文字烙进图片；字体必须覆盖
策划中的人物、地名和法宝字形并允许商业发布。保留语言资源边界和简繁/专名术语表，
为后续英文等本地化留出替换能力，但不要在第一版生成未经文化审校的机器翻译界面。

# 必须保真

## 回合契约

- 五张棋盘均为手工设计，最大不超过 10×12 格；同一状态与行动序列产生同一结果。
- 回合开始显示视线、敌方攻击、火势、风向和同伴意图；提交后的结果必须与预览一致。
- 每个可控单位每回合 2 点行动，移动、变化和能力产生可撤销的玩家阶段计划。
- 提交前可以撤销本回合，提交后只能承担结果或重开当前棋盘；没有命中率、暴击、抽牌、
  随机增援和隐藏范围。
- 虫、蟹、牛魔王、鹰/虎/象分别改变移动、格子权限、受力或角色反应，不能只换模型。
- 金箍棒推位、八戒横耙和前后接力服务位置博弈；普通攻击不能清空棋盘或击杀主要角色。

## 原作因果

- 罗刹女因红孩儿旧怨拒绝借扇；第一次真扇吹走悟空，定风丹使其返回。
- 虫形穿门缝后取得假扇；假扇在火焰山连续三次使火势扩大，真假不能被 UI 提前剧透。
- 土地说明火焰来自悟空当年踢落的八卦炉余火，并指出牛魔王所在。
- 悟空跟踪牛魔王、化蟹潜入碧波潭、观察宴会、偷辟水金睛兽、化牛魔王进入芭蕉洞，
  取得真扇和变大方法，但不知道缩小口诀。
- 牛魔王也会变化并化假八戒反骗；玩家能否识破只改变终局初始状态，不绕过追击。
- 真实八戒与土地接力；鹰、虎、象压缩原作连续相克变化；白牛真形最终由众神协助降伏。
- 真扇前三回合依次息火、生风、落雨，持续四十九扇断绝火根；悟空归还扇子后继续西行。

## 对标边界

- 借 `Invisible, Inc.` 的视野潜入，不借随机地图、养成和永久死亡。
- 借 `Into the Breach` 的敌方意图预告，不借机甲、战役和时间线。
- 借 `Tactical Breach Wizards` 的推位连锁与低惩罚试错，不借枪械、升级和倒带叙事。
- 借 `Hitman GO` 的紧凑棋台可读性，不借棋子美术、暗杀和关卡布局。
- 不做《黑神话：悟空》的低配动作复刻，也不做 `Slay the Spire` 式卡牌构筑。

## 视觉身份

- 使用原创“明代彩印神怪棋台”：等距三维木台、木刻式轮廓、矿物套色和戏台式动作。
- 格子来自砖缝、石阶、水纹和岩层；悬停时才出现辅助轮廓，不常驻发光网格。
- 敌方意图先由角色预备动作表达，再叠加起点、方向和落点；颜色同时配形状与运动。
- 牛魔王身份在世界中没有猴尾、描边或金光破绽；真假扇也使用同一外观与图标。
- 三张签名画面必须来自假扇反火、牛王入洞、三扇成雨的真实回合状态。

# 范围

必须包含：

- 翠云借扇：短推位教学、预告扇风、定风返回、虫形穿缝；
- 假扇反火：三次全局火势变化和可解撤离；
- 碧波偷骑：水流、蟹形、守卫视线、三项观察和坐骑；
- 牛王入洞：主人权限、行为怀疑、零怀疑/风险路线、真扇口诀、假八戒收束；
- 积雷降牛：悟空与八戒双单位、推位接力、三种追猎走法、白牛围困；
- 回访火焰山：息火、生风、落雨、四十九扇、还扇和西路开放；
- 本回合撤销、当前棋盘重开、整局重开和完整结算状态。

明确排除：百回战役、完整七十二变、随机地图、卡组、命中率、装备、技能树、角色升级、
资源经营、自由探索、现代衍生皮肤、可杀死主要角色的分支、多人和后端。

# 实现自由

根据当前环境自行选择最能实现批准设计的框架、渲染方式、棋盘数据结构、寻路、状态
管理、输入映射、原创程序化资产和测试入口。优先做完灰盒五棋盘与确定性结算，再补
彩印材质、动作和音效；不要为了展示技术静默改成实时动作、卡牌或传统血量战棋。

范围压力出现时，依次压缩宴会 NPC 数量、过渡演出、追猎动画和众神模型数量；不得
删除形态走法差异、预览一致性、身份怀疑、真假扇后果和双人围困。

# 完成证据

构建完成后在本文件补充真实运行命令、验证日期和已知限制，并保存以下证据：

- 从新游戏走通五张棋盘、两种假八戒处理、三次真扇和还扇结局；
- 对同一存档状态重复相同行动，证明结果一致且无隐藏随机；
- 对移动、变化、推位、视线、火势和冲锋各抽样验证一次“预览等于结算”；
- 验证每种形态至少有一个其他形态不能完成的走法或权限；
- 验证碧波潭少观察与全观察会产生不同但都可完成的身份路线；
- 验证原形连续推位不能跳过虫关、蟹关、身份关和白牛围困；
- 验证本回合撤销、棋盘重开、整局重开、完成后二次开始和无死局；
- 检查控制台、资源请求、画布非空、动态结算、键鼠输入、目标桌面视口和页面溢出；
- 检查简体中文无缺字、截断、错序和术语漂移，所有玩家可见文案都能从语言资源替换；
- 保存三张签名画面、一张错误预览纠正、一张身份暴露和一张棋盘恢复截图。
