# 来源设定集：《西游记》

## 来源范围

本分析读取仓库内 Project Gutenberg 完整百回本，不用故事梗概、影视印象或单个章回
代替原著。原文是繁体，策划使用简体；证据统一以回目定位。版本、完整性和授权边界
见 `source/SOURCE.md`。

## 世界身份

《西游记》不是孙悟空连续打赢八十一场战斗，而是一支能力、性情和职责互补的队伍，
沿唯一目标不断进入陌生规则：先被表象误导，再查清妖怪、法宝、地缘与天界关系，
最后用变化、智取、协作或求援恢复通路。胜利通常还要解除一地灾难、冤案或失序，
取经队伍才能继续西行。

全书最值得游戏化的能力不是“无限强”，而是悟空把观察所得转成行动：变成能进入
某个空间的形态、冒充掌握关系的人、找到法宝的克制规则、让正确同伴接手，或者意识
到蛮力无法解决当前问题。

## 可追溯的世界规则

| 规则 | 原作事实 | 游戏化边界 |
|---|---|---|
| 西行是一条受约束的路 | 唐僧坚持向有经处前进，火焰山挡住必经之路；绕回无经方向不算完成使命（第59回） | 关卡目标是恢复通路，不能用筋斗云直接跳过整段责任 |
| 变化改变身体与社会身份 | 悟空反复化作虫、鱼、人物或物件进入不同空间；对手也能变化（第2回、第34-35回、第57-58回、第60-61回） | 每种形态必须改变可做动作；变化不是纯皮肤，也不是自动获得对方记忆 |
| 法宝是带规则的工具 | 紫金葫芦、金刚琢、芭蕉扇等各有触发、收放与克制关系（第34-35回、第50-52回、第59-61回） | 玩家通过后果学习规则，不把法宝降格为伤害更高的装备 |
| 信息先于胜负 | 问土地、探洞、辨来历、套口令和寻找主人经常决定下一步（第20-22回、第32-35回及后续多回） | 调查必须改变可用路线、形态或工具，不做无用收集品 |
| 队伍能力互补 | 悟空善变化与侦查，八戒能水战、开路和接力，沙僧守护师父与行李，唐僧维持使命（第19-22回及全书） | 同伴不能只是自动伤害；离位、救援和接手要改变局面 |
| 强敌属于关系网 | 妖怪常与坐骑、童子、法宝主人、龙宫、天庭或佛门相连（第40-42回、第50-52回、第65-66回、第74-77回） | 求援要建立在识别来历后，不能无条件召唤解决剧情 |
| 通关包含地方结果 | 人参果树复活、乌鸡国王复位、祭赛国洗冤、凤仙郡降雨后队伍才离开（第24-26回、第37-39回、第62-63回、第87回） | 胜利画面要显示世界状态改变，而非只弹出“敌人已击败” |

“变化不会自动提供目标的私密记忆”是从多次冒充仍需偷听、套话和维持身份推得的改编
规则，不冒充原文直述。它保留变化术的强大，同时给玩家留下观察和失误空间。

## 主要行动者

- **孙悟空**：玩家候选。好胜、机敏、会犯因得意和急躁造成的错误；核心能力是侦查、
  变化、金箍棒、筋斗云和把更大关系网拉入问题。
- **唐僧**：西行使命和道德约束的持有者。他并非生命条容器；他的判断、恐惧和坚持会
  改变队伍是否继续前进。
- **猪八戒**：贪吃畏难但能水战、开路、激将和关键接力。第61回正是他与土地加入后，
  牛魔王的优势才开始松动。
- **沙僧**：在其他成员出动时保护唐僧、白马和行李，使分队侦查成为可能。
- **白龙马**：龙族身份和坐骑职责并存，不能只当作无生命交通工具（第15回、第30回）。
- **观音与护法体系**：提供使命、辨认来历和少数克制物，但应在玩家已经查明问题后
  出现。
- **地方百姓、土地与国王**：掌握环境信息并承担通关后果，令每一难成为一个具体世界
  而非战斗房间。

## 全书可重复循环

```text
遇到不可直接通过的异象
  -> 向当地人问因并亲自试探
  -> 第一次方案暴露隐藏规则或旧关系
  -> 侦查空间、来历、法宝或人物网络
  -> 选择变化、智取、同伴接力、克制物或求援
  -> 把所得规则用于真实阻碍
  -> 恢复通路与地方状态，继续西行
```

优秀玩家与普通玩家的差别应体现在：是否观察环境入口和人物关系，是否为当前形态选择
正确空间，是否验证拿到的法宝，是否识别对手也在使用变化，以及是否在硬拼无效时改换
办法。

## 可玩动词与空间

- 问路、听取互相矛盾的解释、观察行为与物件，再决定先去哪处。
- 跳跃、攀爬、筋斗云短途穿越和金箍棒格挡，为侦查争取空间。
- 变化为小虫、水族、人物或猛兽，获得缝隙、水下、社交或追逐能力。
- 试用法宝并读取物质后果：风向、火势、雨水、容器吸力或武器失效。
- 与八戒接力、让沙僧留守、向土地或神佛核实来历。
- 在洞府、王城、山路、水域和天界关系网之间建立因果，而不是清空地图图标。

全书空间以“道路上的封闭世界”反复展开：每一地有外部异象、地方聚落、危险边界、
妖怪内域和更高关系出口。这个结构适合短而完整的关卡，不要求第一版做开放世界。

## 视觉与声音锚点

花果山水帘洞、金箍棒伸缩和重量、筋斗云、紧箍、师徒与白马的行旅轮廓、洞府与人间
城郭并置、法宝改变自然现象、妖怪原形与人形转换、天兵云阵、火焰山红热地平线、
芭蕉扇三十六缕红丝、息火生风落雨后的土地复苏。

这些锚点只来自古典文本。任何现代影视的特定甲胄、脸谱、配色、主题音乐和台词都不
属于本示例的视觉依据。

## 选定原型的原作事实

本 demo 改编第五十九至六十一回“三调芭蕉扇”。它不是全书最著名场面的拼贴，而是
用一段连续事件证明全书循环。

| 事实 | 证据 |
|---|---|
| 火焰山横阻西路八百里，四季炎热、寸草不生；当地人依靠铁扇仙定期熄火降雨种粮 | 第59回 |
| 罗刹女即铁扇公主，是牛魔王之妻、红孩儿之母，因红孩儿被观音收为善财童子而怨悟空 | 第59回 |
| 真扇一扇息火、二扇生风、三扇下雨；罗刹女先用扇把悟空吹走，悟空求得灵吉菩萨定风丹后返回 | 第59回 |
| 悟空化虫入腹迫使罗刹女交扇，但拿到的是假扇；假扇连续三次使火势升高 | 第59回 |
| 火焰山来自五百年前悟空蹬倒八卦炉后落下的余火炉砖，火焰也是他旧日行为的后果 | 第60回 |
| 悟空跟踪牛魔王到碧波潭，化螃蟹潜入宴会、偷走辟水金睛兽，再化牛魔王骗得真扇 | 第60回 |
| 真扇缩如杏叶，变大需捻第七缕红丝并念口诀；悟空只问到变大方法，没有问缩小方法 | 第60回 |
| 牛魔王也会七十二变，化作八戒骗回真扇；悟空因得意而没有细察 | 第61回 |
| 真八戒与土地加入追击；悟空和牛魔王连续以飞禽、走兽、法天象地和白牛原形相克 | 第61回 |
| 最终众神协助降伏牛魔王；悟空用真扇三扇息火、生风、落雨，再连扇四十九次断绝火根 | 第61回 |
| 悟空依承诺把扇归还罗刹女，师徒在雨后通过火焰山，当地获得长久生机 | 第61回 |

原文中的强迫入腹、夫妻冒充和围剿情节需要谨慎改编。原型保留“形态取得通路、身份
依赖信息、欺骗会被对手反用”的因果，但不把性暗示或羞辱做成奖励，也不让玩家杀死
罗刹女、牛魔王等原作中存活并归正的角色。

## 改编边界

- `immutable`：火焰山阻路、真假扇后果、罗刹女与红孩儿/牛魔王关系、双方都会变化、
  八戒接力、真扇三次天气变化、四十九扇断根、还扇后西行。
- `adaptable`：跨越数千里的移动时间、宴会潜入长度、众神参战规模、打斗回合数和
  原文成人化社交细节。
- `open`：紧凑关卡的路线、敌人招式、失败检查点、玩家能否提前识破假八戒，以及
  不改变结局的执行质量差异。
- `conflicted`：百回本的作者署名和部分章回文本在学术上有版本讨论；本 demo 只声明
  使用 Gutenberg #23962 所载文本，不把版本判断写进玩法。

完整原著属于公版，不等于所有“西游记 IP”素材都可自由使用。实现必须从文字重新
设计，不复刻任何现代衍生作品。
