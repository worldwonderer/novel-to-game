# 游戏概念：《西游记》

## 产品定位

| 项目 | 决策 |
|---|---|
| 主类型 | 确定性回合制潜入策略 |
| 子类型 | 形态解谜、位置博弈、轻量队友协作 |
| 视角 | 2.5D 等距棋盘，手工关卡 |
| 平台 | 桌面浏览器，单人，鼠标优先并支持键盘 |
| 原作语境 | 明代汉语神魔小说，融合佛教、道教、民间信仰和世俗讽喻 |
| 目标玩家 | 首发面向中文玩家；熟悉原著不是理解规则的前提 |
| 界面语言 | 首发简体中文；英文等其他语言不在第一版范围 |
| 时长 | 18-22 分钟完整可玩原型 |
| 随机性 | 无命中率、暴击和随机掉落；结果可预览 |
| 核心卖点 | 七十二变同时改写走法、可进入区域和社会身份；芭蕉扇直接改写整张棋盘 |
| 市场区隔 | 不做低配西游动作 RPG，也不把变化术压成卡牌和数值增益 |

它不是传统指令战斗 RPG、卡牌构筑、战棋养成、Roguelike 或实时潜行。每个回合都是
“看见世界下一步要做什么，再用正确形态改写结果”的小型因果题。

## 行业对标

| 角色 | 对标作品 | 已被证明的原则 | 本作落地 | 明确不借 |
|---|---|---|---|---|
| 主结构 | `Invisible, Inc.` | 回合制让视野、警戒、精确移动和团队分工成为潜入决策 | 守卫视线、身份限制区、暴露状态和八戒接力全部在格上可预览 | 随机地图、特工养成、黑客程序、永久死亡和企业题材 |
| 回合可读性 | `Into the Breach` | 敌人攻击预告后，玩家用位置和能力改变将发生的结果 | 火势扩张、扇风、牛魔王冲锋和横扫在回合提交前显示落点 | 机甲、随机战役、岛屿成长、时间线叙事和建筑血量 |
| 行动组合 | `Tactical Breach Wizards` | 独特能力、推拉和环境可以形成短而清楚的连锁；允许试错会鼓励实验 | 金箍棒推位、八戒定身、风向和形态走法组合；提交世界回合前可撤销 | 枪械、技能升级、完整倒带叙事和现代战术题材 |
| 空间翻译 | `Hitman GO` | 固定棋盘、禁区、秘密通路、伪装和敌人行为能把潜行压缩成易读回合 | 门缝、水道、主人入口、坐骑和家眷视线构成牛魔王身份关 | 暗杀、棋子美术、逐格击杀和该系列关卡布局 |
| 形态原则 | 《塞尔达传说：姆吉拉的假面》 | 不同形态拥有不同移动能力、可达区域和角色反应 | 虫、蟹、牛魔王、追猎形态分别改变走法与权限 | 面具收集、三日循环、神殿和任天堂角色/UI |
| 世界结果 | 《大神》 | 核心工具能直接改变风、地形和枯败世界 | 真扇连续把火格变为灰格、风路、雨地和永久通路 | 笔触输入、水墨视觉和日本神话角色 |

最近的市场参照是《黑神话：悟空》。它已经占据“西游题材、棍术、法术、变化、强敌”的
高规格动作 RPG 心智，本作必须靠确定性棋盘、身份潜入和环境因果区隔，不能做它的网页
缩小版。`Slay the Spire` 证明卡组构筑本身足以成为完整产品，也因此提醒本作不要把
形态和法宝做成抽卡、费用曲线与遗物组合。

文化与市场证据分开使用：《黑神话：悟空》只证明中文市场会检验西游角色、器物和原作
关系是否可辨识，不证明动作 RPG 是唯一答案；欧美和日本对标只证明潜入、意图预告、
形态走法与世界修复等玩法原则，不作为中国神话、美术或人物关系的依据。首发中文界面
保留“变化”“定风丹”“辟水金睛兽”等原作术语，通过棋盘后果和短提示让未读原著的玩家
理解，不替换成另一文化的通用魔法、药水或坐骑标签。

最终配方是：**`Invisible, Inc.` 的信息潜入 + `Into the Breach` 的意图预告 +
`Tactical Breach Wizards` 的位置连锁**，再用《姆吉拉的假面》约束每个形态必须改变
走法，用《大神》约束结局必须写回世界。

## 方向一：三借芭蕉扇

**一句话核心卖点**：玩家在小棋盘上预读火势、守卫与牛魔王的下一步，变化成虫、蟹、
牛魔王或追猎形态，改变走法和身份权限，最终用真扇永久重写火焰山。

- **玩家身份**：第五十九至六十一回的孙悟空；终局短暂加入可控制的八戒。
- **核心动词**：移动、观察、变化、推位、潜入、冒充、扇动、提交回合。
- **回合循环**：读取意图与视线 → 选择形态和位置 → 预览结果 → 提交世界回合 →
  从实际后果修正下一步。
- **压力**：火格每回合扩张，身份错误积累怀疑，牛魔王冲锋压缩安全位；没有随机命中
  替玩家承担判断。
- **熟练度差异**：新玩家逐个处理威胁，熟练玩家用一次变形或推位同时解决通路、视线
  和下一回合站位。
- **原作转换**：真假扇需要在世界上试验，牛魔王用同一变化规则反骗，八戒的接力改变
  可用行动，法宝最后改变地方生计（第59-61回）。
- **传播画面**：一张等距火焰山棋盘上，假扇的风箭本应压灭火线，却把三列火格推向
  师徒；悟空必须在提交回合前找到唯一不会抛下同伴的撤离链。
- **原型范围**：五张不超过 10×12 格的手工棋盘，18-22 分钟；不做百回战役、装备、升级、
  卡组或随机地图。
- **最大风险**：如果每个形态只有“允许进入某色格”，会退化成钥匙锁；必须让走法、
  风险和连锁也发生变化。

## 方向二：西行驿路

**类型**：回合制行旅管理；俯视路线图与队伍面板。

玩家作为唐僧，每回合在一日路程内派悟空侦查、八戒找食、沙僧守护或全队前行。报告
并不完整，信任、饥渴、日落和关文共同变化。核心不是单关解谜，而是承担“最强者离队
后谁保护队伍”的长期后果。

- **对标原则**：借《The Banner Saga》的行旅压力与队伍关系，不借其战斗、世界和
  资源模型。
- **原型范围**：白骨精至宝象国的一次离队/召回闭环（第27-31回）。
- **传播画面**：四条行路任务同时倒数，悟空已经离队，唐僧必须在八戒的错误报告与
  沙僧的守备警告之间决定下一回合。
- **淘汰原因**：真正价值依赖数小时关系积累；20 分钟 demo 容易变成事件按钮和数值条。

## 方向三：二心

**类型**：同步指令回合制护卫解谜；单场多阶段棋盘。

玩家作为沙僧，先为两个外观、能力和意图都相同的悟空规划行动，再同时结算，目标不是
猜中 UI 中的“真身”，而是保护唐僧、白马和西行通路不被两人的冲突摧毁（第57-58回）。

- **对标原则**：借 `Frozen Synapse` 的同步计划与结果预演，不借其枪战和程序地图。
- **原型范围**：花果山、水帘洞、地府与灵山三次辨认，每处改变能保护的空间规则。
- **传播画面**：两根金箍棒的攻击线同时覆盖棋盘，沙僧只能移动一人并用降妖宝杖改变
  另一条线，唐僧身后的西路只剩两格完整。
- **淘汰原因**：原著让谛听知情却不敢说，最后由如来辨明；若游戏承诺玩家推理出真身，
  就必须虚构原作不存在的可靠破绽，若不承诺又缺少结局能动性。

## 选择状态

选择 **三借芭蕉扇**。它用连续三回提供五种回合问题：环境蔓延、微型潜入、水下走法、
社会身份和双人围困；所有问题仍由同一个“预读后变化”循环解决。

不可妥协项：

- 敌方意图、火势和行动结果在提交前可读，失败来自判断而非骰子。
- 形态改变走法、权限和风险，不能只提供伤害倍率。
- 假扇取得时不可被 UI 剧透，必须通过火焰棋盘确认。
- 牛魔王必须使用同一变化规则反骗；八戒必须作为不同走法的第二单位加入终局。
- 金箍棒和钉耙主要推位、打断和固定，不做清空敌人的伤害竞赛。
- 结束包含熄火、生风、落雨、断根、还扇和继续西行。
