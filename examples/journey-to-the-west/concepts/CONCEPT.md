# 游戏概念：《西游记·三借芭蕉扇》

## 产品定位

| 项目 | 决策 |
|---|---|
| 主类型 | 回合制指令 RPG（《梦幻西游》《问道》风） |
| 子类型 | 五行相克对战、召唤兽携宠、阵型、七十二变、多阶段 Boss、升级加点与法术熟练、剧情装备与规则型法宝、物品背包 |
| 视角/呈现 | 全屏战斗舞台，敌方左上斜列、我方右下斜列的经典对阵 |
| 平台 | 桌面浏览器（≥1280×800），单人，鼠标优先并支持键盘 |
| 原作语境 | 明代汉语神魔小说，融合佛教、道教、民间信仰和世俗讽喻 |
| 目标玩家 | 熟悉「回合制西游」的中文玩家；不熟原著也能上手 |
| 界面语言 | 首发简体中文；其他语言不在第一版范围 |
| 时长 | 约 45-90 分钟的完整单人战役（序幕 + 第五十九至六十一回九段可玩桥段 + 结局） |
| 随机性 | 命中、暴击、五行相克、伤害浮动都有概率，但用 seedRNG（mulberry32）固定种子保证可复现 |
| 核心卖点 | 中文玩家一眼就认出的回合制西游语法（速度行动顺序、指令出招、五行相克、携宠协战、阵型、七十二变）＋ 明代木刻套印的美术辨识度 ＋ 忠于原著「三借芭蕉扇」 |
| 市场区隔 | 借回合制西游玩法心智，不做动作 RPG，也不做抽卡、门派数值、PVP 和 MMO 长线养成 |

它正是中文玩家最熟的那类回合制指令 RPG——不是动作 RPG、卡牌构筑、Roguelike 或
战棋养成。每回合的核心是“为全队下达指令，按速度从高到低逐个结算，用五行相克、
携宠、阵型与变化在正确的回合翻盘”，最终以真扇三段扭转牛魔王之战。

## 行业对标

| 角色 | 对标作品 | 已被证明的原则 | 本作落地 | 明确不借 |
|---|---|---|---|---|
| 主结构 | 《梦幻西游》 | 指令回合制＋速度行动顺序＋五行相克，是中文玩家最熟的战斗语法 | 每回合先为全队下指令，按速度降序生成含敌方的行动队列，逐个结算并飘出伤害与「克!」 | 抽卡、点卡经济、帮派、师门与 MMO 长线养成 |
| 相克与养成 | 《问道》 | 五行金木水火土循环相克＋轻量升级加点，一眼就懂 | 金克木、木克土、土克水、水克火、火克金；每胜一场全员升级、按等级解锁法术 | 门派数值墙、装备打造、交易与 PVP |
| 携宠与协战 | 《梦幻西游》《大话西游》 | 携带宠物作为独立参战单位，有自己的指令与技能 | 辟水金睛兽（水）作为第四单位入队，辟水诀水系群攻克火、金睛提升全队命中 | 宠物抽卡、繁殖、洗练等养成经济 |
| 阵型与变化 | 《大话西游》《梦幻西游》 | 阵型改变攻防速，变化术临时改写角色能力 | 天罡阵偏攻、六丁阵偏防；悟空「七十二变」三回合改五行、属性与技能 | 数十套阵型/变身的收集养成深度 |
| 题材辨识 | 《黑神话：悟空》 | 中文市场会检验西游角色、器物与原作关系是否可辨识 | 火焰山、芭蕉扇、罗刹女、牛魔王、辟水金睛兽均忠于原著呈现 | 动作 RPG 形态、高规格实时战斗与硬核操作 |

《黑神话：悟空》证明西游题材有高规格市场，但它占据的是“棍术、法术、变化、强敌”的
动作 RPG 心智。本作不做它的网页缩小版：我们是回合制指令、五行相克与明代木刻美术，
靠一眼可辨的回合语法与视觉身份区隔，而非比拼实时操作。

《梦幻西游》《问道》《大话西游》定义了中文玩家的回合制西游心智——回合指令、速度序、
五行相克、携宠、阵型、变化都是被反复验证的语法。本作原样沿用这套熟悉玩法，让没读过
策划案的玩家也能上手；但明确不借抽卡、门派数值墙、PVP、经济系统与 MMO 长线养成。
这是一条约 45-90 分钟、忠于原著第五十九至六十一回的完整单人战役，不是需要长期投入的网游。

玩法心智与文化依据分开使用：战斗骨架借自国产回合制西游，人物、器物与事件依据只来自
古典文本第五十九至六十一回。界面保留“芭蕉扇”“辟水金睛兽”“定风丹”“七十二变”等
原作术语，通过战斗后果与短提示让未读原著的玩家理解，不替换成他文化的通用魔法、
药水或坐骑标签。

最终配方是：**《梦幻西游》《问道》的指令回合制骨架（速度序＋五行相克＋携宠＋阵型＋
变化）＋ 明代木刻套印的视觉辨识度 ＋ 忠于原著的「三借芭蕉扇」战役**。用 seedRNG 让
概率结算可复现（自检可重放），用多阶段 Boss 与真扇三段提供清晰的翻盘节奏。

## 方向一：三借芭蕉扇

**一句话核心卖点**：玩家指挥孙悟空一行，用指令回合制在速度序里出招——五行相克、
携宠协战、阵型切换、七十二变层层叠加，最终以真扇三段翻盘牛魔王，息火、生风、落雨后
还扇西行。

- **玩家身份**：第五十九至六十一回的孙悟空一队（悟空、八戒、沙僧，火焰山起携辟水
  金睛兽协战）；唐僧随队不参战。
- **核心动词**：下指令、按速度序结算、攻击、施法、防御、用道具、变化、切阵型、动用
  芭蕉扇、自动代打、逃跑（非 Boss）。
- **回合循环**：为每位伙伴下达指令 → 生成速度降序行动队列（含敌方，见顶部行动顺序
  条）→ 逐个结算并飘字（伤害/暴击/克） → 死亡与胜负判定 → 下一回合。
- **战斗结算**：属性含 HP/MP/攻击/防御/速度/灵力＋五行；伤害 = 攻×技能倍率 − 防×0.5，
  再乘五行系数（克 1.5 / 被克 0.66 / 中性 1）× 暴击 × 浮动，下限为 1；一切随机走
  seedRNG，飘字显示实际数值与「暴击/克」。
- **相克体系**：金克木、木克土、土克水、水克火、火克金。悟空（金）压罗刹女（木），
  辟水金睛兽（水）灭火兵（火），八戒（木）克白牛真身（土）；而牛魔王人形属火反克
  悟空（金），须靠携宠或变玄甲龟将（水）压制——相克贯穿全程且在 Boss 战反转。
- **携宠**：辟水金睛兽为第四参战单位，有独立指令与技能——辟水诀水系群攻克火、
  金睛提升全队命中三回合。
- **阵型**：天罡阵（攻+12% 速+5%）与六丁阵（防+18% 受伤-10% 速-5%），战斗内外均可
  切换，战斗内切换消耗该单位本回合行动。
- **变化（悟空专属特技）**：七十二变三回合改五行与技能——金甲神将（攻+40%，得群体
  物理）、赤焰灵猿（灵力+45%，得火系法术）、玄甲龟将（防+60% 速-20%，得水系法术）、
  蟭蟟虫（速+50% 攻-20%，抢速自保）。
- **芭蕉扇**：假扇一扇火更旺，反噬使火敌攻击提升；真扇三段——一息火（清敌增益、火敌
  减攻）、二生风（全队加速）、三落雨（全队持续回血、火敌破防），是 Boss 战的翻盘节点。
- **升级养成**：每场胜利升级并发潜力点，自由投 体/攻/防/速/灵；法术随等级熟练进阶（威力升、耗蓝降）；
  收服召唤兽（剧情门控捕捉，如辟水金睛兽、火妖赤焰火骝）上阵协战；剧情掉落的简单武器装备与规则型
  法宝（定风丹免疫减速、避火锦抗火，不加攻）；物品背包（丹药/捕妖绳/芭蕉扇）。**不做**抽卡、门派数值墙、
  装备打造与交易、PVP 或 MMO 长线养成。
- **战役范围**：序幕小世界 + 第五十九至六十一回九段可玩桥段（罗刹女一扇吹飞→灵吉授定风丹→化虫取假扇→
  假扇反火→摩云洞玉面公主→初战牛魔王→碧波潭变螃蟹偷金睛兽→骗真扇→反骗→积雷山决战+众神围剿+白牛真身+
  真扇三段）+ 还扇西行结局，约 45-90 分钟；不做百回战役、开放世界、卡组或随机地图。
- **原作转换**：假扇需通过战斗后果（火越扇越旺）确认真假；牛魔王同样会七十二变、
  化作八戒反骗，直到真八戒与土地加入，牛魔王的优势才开始松动（第 61 回）。
- **传播画面**：牛魔王残血化白牛真身、体量压场，全队濒危之际悟空取出真扇——一息火、
  二生风、三落雨连翻战局，雨落火焰山，师徒得以过山西行。
- **最大风险**：若变化与阵型只提供伤害倍率而不改变战术选择，会退化成数值堆叠；必须
  让相克、携宠、变化与真扇形成清晰、可读的战术连锁。

**战役节拍**（原型实装五段）：

- **序幕·火焰山脚**：斜视小世界场景，点击移动、问土地得知“火焰山八百里、需借芭蕉扇
  息火”，走向翠云山进入战斗。
- **战斗一·翠云山·芭蕉洞外（教学）**：对手罗刹女（木，约 980 体力）。教指令菜单、
  速度序、五行相克、暴击飘字与悟空变化；罗刹女体力≤55% 时悟空变化可触发「化虫入腹」
  直接取胜（原著第 59 回彩蛋）。胜利得假扇，界面不剧透真假。
- **战斗二·火焰山·火口（携宠）**：假扇使火更旺，火兵×2＋火炎校尉（火）成群来袭。
  教水克火、法术群攻与携宠辟水金睛兽参战；得线索“真扇在牛魔王处”。
- **Boss·积雷山·摩云洞前（两阶段）**：牛魔王（火，约 1500 体力）人形阶段 → 残血化
  白牛真身（土，约 1950 体力）变大压场；由协战＋携宠＋变化＋真扇翻盘。相克在此反转：
  火形反克悟空需水系压制，土形则交八戒（木）克制。
- **结局·还扇西行**：真扇一息火、二生风、三落雨，悟空依约把扇归还罗刹女，师徒雨后
  通过火焰山，当地重获长久生机。

## 方向二：西行驿路

**类型**：回合制行旅管理 RPG；俯视路线图与队伍面板。

玩家作为唐僧，每回合在一日路程内派悟空侦查、八戒找食、沙僧守护或全队前行。报告并不
完整，信任、饥渴、日落和关文共同变化。核心不是单场战斗，而是承担“最强者离队后谁
保护队伍”的长期后果。

- **对标原则**：借《The Banner Saga》的行旅压力与队伍关系，不借其战斗、世界和资源模型。
- **原型范围**：白骨精至宝象国的一次离队/召回闭环（第27-31回）。
- **淘汰原因**：真正价值依赖数小时关系积累；15-22 分钟切片容易退化成事件按钮和数值条，
  也发挥不出回合制西游最被认可的对战语法。

## 方向三：二心

**类型**：真假辨认回合制护卫解谜；单场多阶段。

玩家作为沙僧，为两个外观、能力和意图都相同的悟空排布回合指令，目标不是猜中 UI 里的
“真身”，而是保护唐僧、白马和西行通路不被两人的冲突摧毁（第57-58回）。

- **对标原则**：借双单位真假对峙的紧张感，不借枪战题材与程序地图。
- **原型范围**：花果山、水帘洞、地府与灵山三次辨认，每处改变能保护的空间规则。
- **淘汰原因**：原著让谛听知情却不敢明言，最终由如来辨明真假；游戏若承诺玩家自行
  推理出真身，就得虚构原著不存在的可靠破绽，若不承诺又缺少结局能动性。

## 选择状态

选择 **三借芭蕉扇**。它用连续三回提供一条完整的回合制战役：教学对战、群怪携宠、
两阶段 Boss 与真扇翻盘，全部由同一套“下指令 → 速度序结算 → 相克与变化取舍”的
回合语法承载，是中文玩家一眼就认得的回合制西游。

不可妥协项：

- 原著事实为准：火焰山横阻八百里、真扇三段（息火/生风/落雨）与假扇越扇火越旺、
  罗刹女是牛魔王之妻与红孩儿之母、牛魔王也会七十二变并化八戒反骗、四十九扇断火根、
  还扇后西行（第 59-61 回）。
- 五行相克成体系：金克木、木克土、土克水、水克火、火克金，克制 ×1.5、被克 ×0.66，
  命中弹「克!」；相克贯穿队伍配置与 Boss 两阶段的解法。
- 指令回合制骨架不可省：速度降序行动队列、法术耗 MP、防御减伤并回蓝、道具与特技菜单、
  飘字显示暴击与克制。
- 携宠、阵型、七十二变三者都要在，且各自改变战术选择，而非只叠加伤害倍率。
- 多阶段 Boss：牛魔王人形 → 白牛真身两阶段，真扇三段是设计好的翻盘节点。
- 随机性用 seedRNG 固定种子，保证同种子同操作可复现、自检可重放，但不追求消除概率
  ——命中、暴击、相克与浮动仍是回合制西游的手感来源。
- 忠于原著，不做抽卡、门派数值、PVP、经济系统与 MMO 长线养成；不杀死罗刹女、牛魔王等
  原著中存活并归正的角色。
